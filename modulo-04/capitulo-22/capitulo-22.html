iframe

Iframe o significado em, grosso modo, inline frame, traduzindo seria algo como quadro em linha. Iframe é o link dos vídeos que pegamos no youtube. É como se tivesse uma janela para exibir outro site dentro do seu site.

Porém nem todos o sites liberam o uso de seus links por iframe por questões de segurança. Sim iframe apresentava problemas na segurança do site em versões anteriores do html. Mas no HTML 5 isso não é uma problema pois temos como nos previnir de possíveis ataques.

Exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit...

Veja que ele ficou exatamente um quadro na linha.

O comando do iframe é parecido com o âncora (link).Porém o que fica entre as tag no âncora é o nome do link ja no iframe ele só exibirá o que está entre as tag se o navegador no for compatível com iframe.

Por isso é muito comum colocarmos um link para o site do iframe entre essas tag, pois assim caso não apareça não perderá o sentido no texto. Veja como ficou o exemplo de cima abaixo:

Lorem ipsum dolor sit <iframe src=""../../indice de links .html"" frameborder="1"><a href="../../indice de links .html"> Índice</a></iframe>amet consectetur adipisicing elit...

Navegação no iframe

Uma forma interessante de se trabalhar com iframe é utilizando ele como um mini navegador. No caso você não vai digitar uma url, mas pode colocar link que invés de abrir em outra pag com o atributo target:"_blank", ele abrirá dentro do iframe:

Veja um exemplo abaxo:

  • Capítulo 19
  • Capítulo 20
  • Capítulo 21

o Funcionamento é bem simples para isso basta colocar um atributo "name=""" no iframe e colocar um nome qualquer em name para referêciar esse iframe. o nome usado no exemplo acima foi tela.

Feito isso no link que queremo que abra no Iframe basta colocar em target o name que colocou no iframe.

Veja o código usado exemplo acima:

Código no link:

<a href="../../../modulo-03/capitulo-19/capitulo-19.html" target="tela">Capítulo 19

Código no iframe:

<iframe class="tela" name="tela"src="" frameborder="1"></iframe>

Segurança

O iframe possui algumas falhas de segurança como podemos ver na aba desvantagens abaixo. Porém tem como impedir links externos tenham acesso a nossos dados com alguns atributos simples como :

  • sandbox="sandbox" Esse atributo impede que qualquer informação seja enviada para o iframe. Entretanto se o usário precisar interagir com o iframe ele não consiguirá nem fazer um login. Ele conseguirá digitar mas não conseguirá enviar esses dados.

    Porém é possivel liberar alguma permições como:

    sandbox="sandbox" Bloqueia tudo. Recomendado usar esse sempre que levar para link externo
    sandbox="allow-same-origin" Permite ao usuário interagir com conteúdos de mesma origem, ou seja link do mesmo site.
    sandbox="allow-forms" Permite o preenchmento e envio de formulários dentro do iframe
    sandbox="allow-scripts" Permite a execução de java script dentro do iframe [arriscado, tenha muita confiança nesse site]
  • referrerpolicy="no-referrer"
    • isso impede que os sites dentro do iframe colete qualquer dado do usuário como cookies ou qualquer outra coisa por mais simples que seja.

Melhores utilidade do iframe

Como ja vimos anteriormente o iframe pode ser usado para incorporar vídeos do youtube e do vímeo. No entanto temos outros serviços do Google que permitem a incorporação de de conteúdo via iframe e alguns deles são:

  • Google Maps Aqueles mapa do Google Maps que aparecem em alguns sites para clicar é iframe para isso basta ir no site Google Maps → ir na localização que deseja no mapa → clicar em compartilhar → e clicar em incorporar. Assim irá aparecer um iframe igual ao que aparece no youtube.
  • Waze Sim o waze também da pra incorporar o procedimento é igual ao do Google maps. Mas o waze tem a vantagem de mostrar o trânsito no local em tempo real, além de outros alertas.
  • Documentos Google Também é possível incorporar Documentos Google no iframe, esse ainda não testei mas parece ser interessante. O exemplo em aula foi uma apresentação de slide(PowerPoint)

Obeservações

Um dos principais uso de iframe é para dar uma prévia de um link do próprio site.

Desvantagens do uso de iframe

  • Navegadores imcompatíveis Muitos navegadores são imcompatíveis com iframe, principalmente navegadores Mobile.
  • Link fora do ar Caso o link para o iframe saia do ar ele não aparecerá o que está entre a as tag iframe. ele só faz isso caso o navegador seja incompatível
  • Segurança Caso seu iframe tenha um link para um site externo e esse site for invadido e comecem a roubar dados do dele, o seu site também terá seus dados roubados. O iframe cria uma ligação direta com o outro site.
  • SEO O Google já anunciou que as vezes tem alguns problemas para indexar iframes ao site. Isso é, ele não consegue verificar o que há no iframe, sendo assim, mesmo que o conteúdo seja relevante ele não exibirá nas pesquisas dele, pois ele não reconheceu que seu site possui aquele conteúdo
  • Acessibilidade Pessoas com deficiência visual utilizam leitores de telas e esses leitores costumam apresentar erro ao ler o que está no iframe
  • Navegação Prejudicada Como vimos, é possível navega em site pelo iframe, porém temos alguns problemas como quando precisa voltar a página do iframe nem sempe o botão de volta do navegador funciona no iframe e quando funciona pode confundir o usuário que esta tentando voltar a página interia e ele fica o iframe. Outro problema é abrir link no Iframe que pode tanto abri em uma no guia ou no próprio iframe.
