Introdução ao Iframe

Iframe, ou "inline frame", é um elemento HTML que permite a incorporação de outro documento HTML dentro de uma página web. Um exemplo comum é a incorporação de vídeos do YouTube.

Por questões de segurança, nem todos os sites permitem que seus conteúdos sejam exibidos em iframes. Embora o uso de iframes apresentasse problemas de segurança em versões anteriores do HTML, o HTML5 introduziu mecanismos para mitigar esses riscos.

Exemplo de Uso

Vamos ver um exemplo de uso do iframe:

O código abaixo incorpora um documento HTML dentro da página atual:

Lorem ipsum dolor sit amet consectetur adipisicing elit...

Note que o iframe aparece como um quadro embutido na linha do texto.

O comando do iframe é semelhante ao de um link (âncora). No entanto, o conteúdo entre as tags do iframe só será exibido se o navegador não suportar iframes. Por isso, é comum colocar um link para o site do iframe entre essas tags, garantindo que o texto ainda faça sentido caso o iframe não seja exibido.

Veja como ficou o exemplo acima:

Lorem ipsum dolor sit <iframe src=""../../index.html"" frameborder="1"><a href="../../indice de links .html">Índice</a></iframe> amet consectetur adipisicing elit...

Navegação no Iframe

Uma maneira interessante de usar iframes é como um mini navegador. Você pode criar links que abrem dentro do iframe, em vez de abrir em uma nova aba com o atributo target="_blank":

Veja o exemplo abaixo:

  • Capítulo 19
  • Capítulo 20
  • Capítulo 21

Para isso funcionar, adicione um atributo name ao iframe e use esse nome no atributo target dos links. No exemplo acima, o nome usado foi "tela".

Veja o código usado no exemplo acima:

Código do link:

<a href="../../../modulo-03/capitulo-19/capitulo-19.html" target="tela">Capítulo 19</a>

Código do iframe:

<iframe class="tela" name="tela" src="" frameborder="1"></iframe>

Inicialização do Iframe

Para o iframe não iniciar em branco, você pode fazer uma das seguintes opções:
  1. Colocar uma URL

    2. Adicione a URL de um dos links no atributo src do iframe para que ele já inicie com esse conteúdo. Ao clicar em outro link, o conteúdo será substituído.
  1. Adicionar texto ou imagem

    2. Substitua o src="" por srcdoc="" e escreva o conteúdo desejado. Você pode usar tags HTML, incluindo imagens. Para imagens, use aspas simples no atributo src da tag img.

    Exemplo: <img src='endereço' alt="">
  1. Criar uma página dedicada

    2. A melhor opção é criar uma página simples e bem feita, que pode ser usada em vários iframes e será mais leve que uma imagem.

Segurança

O iframe possui algumas falhas de segurança, mas podemos mitigar esses riscos com alguns atributos:

  • sandbox="sandbox" - Esse atributo bloqueia todas as interações com o iframe. No entanto, você pode liberar algumas permissões específicas:
    sandbox="sandbox" Bloqueia tudo. Recomendado para links externos.
    sandbox="allow-same-origin" Permite interações com conteúdos da mesma origem.
    sandbox="allow-forms" Permite o preenchimento e envio de formulários.
    sandbox="allow-scripts" Permite a execução de JavaScript (use com cautela).
  • referrerpolicy="no-referrer" - Impede que sites dentro do iframe coletem dados do usuário, como cookies.

Melhores Utilidades do Iframe

O iframe pode ser usado para incorporar vídeos do YouTube e Vimeo, além de outros serviços do Google:

  • Google Maps - Para incorporar um mapa, vá ao Google Maps, selecione a localização, clique em compartilhar e depois em incorporar.
  • Waze - Similar ao Google Maps, mas com informações de trânsito em tempo real.
  • Documentos Google - Você pode incorporar documentos do Google, como apresentações de slides.

Um dos principais usos do iframe é para dar uma prévia de um link do próprio site.

Desvantagens do Uso de Iframe

  • Navegadores incompatíveis - Alguns navegadores, especialmente em dispositivos móveis, podem não suportar iframes.
  • Link fora do ar - Se o link do iframe estiver fora do ar, ele não exibirá o conteúdo entre as tags iframe.
  • Segurança - Se o site externo do iframe for comprometido, seu site também pode ser afetado.
  • SEO - O Google pode ter dificuldades para indexar o conteúdo do iframe, afetando a visibilidade nas pesquisas.
  • Acessibilidade - Leitores de tela podem ter problemas para ler o conteúdo do iframe.
  • Navegação Prejudicada - A navegação dentro do iframe pode ser confusa, especialmente com o botão de voltar do navegador.
