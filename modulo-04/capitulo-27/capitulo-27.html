Flex Box

O que são flex Box?

Flex box são caixas flexíveis.

A ideia inicial do uso das flex box é ajustar o conteúdo do site de acordo com cada formato e tamanho de tela disponível no mercado. Elas funcionam da seguinte forma:

A caixa externa dita as regras de tamanho disponível para exibição. Já as internas se moldam ao tamanho da externa de acordo com as regras que impusermos a elas. O objetivo é que elas sempre trabalhem de forma fluida.

Para deixar as boxes com essas características, precisamos usar sempre 2 comandos:

  • display: flex; Usamos display flex na box externa, mais conhecida como container.
  • flex: auto; Usamos flex auto nas boxes internas, ou caixas filhas.

Eixos (axis)

Além das direções da caixa contêiner, existem ainda mais dois eixos dentro dela:

  • main-axis: Como o nome já diz, este é o eixo principal, o eixo mais longo do retângulo. É o eixo que atravessa o contêiner na direção da flex-direction. Resumidamente seria o eixo X do plano cartesiano.
  • cross-axis: Também como o próprio nome já diz, este cruza perpendicularmente o eixo principal (main-axis). Resumidamente seria o eixo Y do plano cartesiano.

Flex-direction

Flex-direction é uma propriedade que altera o eixo principal e a direção dos eixos, em outra palavras ela pode traformar o eixo Y no eixo principal e trocar os lados negativo e positivo do plano cartesiano.

Alinhamento de conteúdo empacotado (align-content)

A propriedade align-content é utilizada para alinhar as linhas de itens flexíveis dentro de um contêiner flexível quando há espaço extra no eixo transversal (cross-axis). Ela é aplicada somente quando os itens flexíveis são empacotados em várias linhas, ou seja, quando a propriedade flex-wrap está definida como wrap ou wrap-reverse. A align-content define como o espaço extra entre as linhas de itens flexíveis deve ser distribuído, permitindo um controle mais preciso sobre o layout do contêiner.

Autonomia dos itens flex-box

Por padrão os itens do conteiner não são flexíveis, isso quer dizer que eles apenas obedecem a organição do conteiner. Entretanto, tem como mudar o comportamento individual de cada item para que cada um se organize do seu jeito dependendo do formato do conteiner.

order

Order é uma forma de organizar os itens dentro de um contêiner. Por padrão, todo item do contêiner recebe order=0, sendo assim a ordem que escrevemos a div dentro do contêiner é a ordem que os itens ficarão organizados. Porém, nem sempre isso é viável dependendo do tamanho do código, por isso existe o order.

Exemplo:

1
2
3
4
5

Comando: 

<div class="contex19">
    <div class="intex" style="order: 5">1</div>
    <div class="intex" style="order: 4">2</div>
    <div class="intex" style="order: 3">3</div>
    <div class="intex" style="order: 2">4</div>
    <div class="intex" style="order: 1">5</div>
</div>

align-self

Esta propriedade de estilo tem como principal finalidade dar autonomia às caixas internas de um contêiner. Até aqui, vimos todas as caixas trabalhando iguais, apenas seguindo as regras do contêiner. Mas agora, com esta propriedade, poderemos dar estilos independentes para as caixas internas.

Para isso, usaremos 5 estilos:

  • auto ou inherit: Este é o estilo padrão, portanto ele seguirá as regras do contêiner.
  • flex-start: Este alinhamento faz com que a caixa vá para perto do início do eixo transversal (cross-axis).
  • flex-end: Este, naturalmente, faz a caixa ir para o final do eixo transversal (cross-axis).
  • center: Centraliza a caixa na metade do eixo transversal (cross-axis), em outras palavras, coloca a caixa em cima do eixo principal (main-axis).
  • stretch: Estica a caixa de uma ponta a outra do eixo transversal (cross-axis).

flex-basis

A propriedade flex-basis define o tamanho inicial de um item flexível antes de qualquer espaço extra ser distribuído ou qualquer ajuste de tamanho ocorrer. Ela pode ser definida em qualquer unidade de medida CSS, como pixels (px), porcentagem (%), em, rem, etc.

Se a propriedade flex-basis não for especificada, o valor padrão é auto, o que significa que o tamanho do item será determinado pelo seu conteúdo.

Você pode usar a propriedade flex-basis em conjunto com flex-grow e flex-shrink para criar layouts flexíveis e responsivos.

As propriedades flex-grow e flex-shrink são usadas para controlar a capacidade de um item flexível de crescer ou encolher dentro de um contêiner flexível.

flex-grow

A propriedade flex-grow define a capacidade de um item flexível de crescer para ocupar o espaço disponível no contêiner. O valor padrão é 0, o que significa que o item não crescerá além do seu tamanho inicial. Um valor maior que 0 permite que o item cresça proporcionalmente ao espaço disponível.

Propriedade flex

A propriedade flex é uma abreviação para as propriedades flex-grow, flex-shrink e flex-basis. Ela permite definir o comportamento de crescimento, encolhimento e tamanho base de um item flexível em uma única linha de código.

Exemplo de uso:

Comando:

Estilos Funcionamento e Aplicações
flex: 0 1; Esses dois comandos resetam a box para os padrões iniciais. Isso é usado para evitar que algum estilo generalizado interfira no funcionamento esperado da box interna.
flex: initial
flex: 0 0 auto; Esses comandos removem a flexibilidade da box e a deixam apenas no tamanho do conteúdo delas.
flex: none;
flex: 1 1 auto; Esses comandos fazem o inverso dos anteriores. Eles deixam a box 100% flexível e ajustam seu conteúdo interno também.
flex: auto;
flex: 1; Quando há apenas um número, significa que só foi definido o flex-grow. Logo, o flex-shrink e o flex-basis assumem o valor padrão. Seria o mesmo que escrever flex: 1 1 auto;
