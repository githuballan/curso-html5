Flex Boxes

Flex box são caixas flexíveis. Valeu até a próxima!

A ideia inicial do uso das flex box é ajustar o conteúdo do site de acordo com cada formato e tamanho de tela disponível no mercado. Elas funcionam de seguinte forma:

A caixa externa dita as regras de tamanho disponível para exibição. Já as internas trabalham se moldando ao tamanho da externa de acordo com a regras que impulsermos a elas. O objetivo é que elas sempre trabalhem de forma fluida.

Para isso deixar as boxes com essa caracteríscas precisamos usar sempre 2 comandos:

  • display: flex; Display flex usamos na box externa. Mais comnhecida com conteiner
  • flex: auto; A flex auto usamos nas box internas. Caixas filhas.

<div id="conteiner">
    <div id="item01">BOX 01</div>

    <div id="item03">BOX 03</div>
    <div id="item04">BOX 04</div>
    <div id="item04">BOX 05</div>
    <div id="item04">BOX 06</div>
</div>

Comando no head:

                       
                
<style>
    div#conteiner{
        display:flex;
        background-color: #990000;
        text-align: center;
        flex-wrap: wrap;
    }

    div#item01, div#item02, div#item03, div#item04,div#item05, div#item06{
        flex:auto;
        background-color: #0000ee; 
        border: solid;
        border-radius: 5px;

        margin:3px;
        color:#ffffff;
        padding: 1%;
        
    }
        
</style>

Flex-direction

Flex-direction é um estilo usado na box conteiner. Como o nome já diz, ela serve para direcionar de onde para onde as box internas deverão se organizar.

Existem 04 direcionamentos possíveis:

  • flex-direction: row; Nessa forma as box internas iram se alinhar na horizontal da esquerda para a direita.
  • flex-direction: row-reverse; Nessa forma as box internas iram se alinhar na horizontal da direita para esquerda.
  • flex-direction: column; Nessa forma as box internas iram se alinhar na vertical de de cima para abaxo.
  • flex-direction: column-reverse; E nessa forma as box internas iram se alinhar na vertical de de baixo para acima.

Eixos (axis)

Além das direção da box conteiner, existe ainda mais dois eixos dentro dela:

  • main-axis Como o nome já diz esse é o eixo principal ele o eixo mais longo do retângulo. É eixo que atravessa o conteiner na direção da flex-direction.
  • cross-axis Também como o próprio nome já diz, essa cruza o perpendicularmente o eixo principal(main-axis).

Direcionamentos de cross-axis

Direcionamentos cross-axis são simples pois eles só quando a flex-direction está na horizontal(row) ou na vertical(column), veja:

  • O cross-axis, em casos de flex-directions horizontais(rows), Sempre serão de cima para baixo. Como o padrão column.
  • E quando a flex-direction está na vertical ele será sempre da esquerda para dereita. Como a row padrão.
