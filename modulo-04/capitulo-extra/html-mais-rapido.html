Atalhos para abreviar o códigos

  • Uma tag dentro da outra: Para adicionar tags em sequência uma dentro da outra basta escreve a tag mais externa seguida do sinal de > e interna, veja o exemplo abaixo
    •                     main>div>p

    ficaria da segunite forma:

    
                    <main>
                        <div>
                            <p></p>
                        </div>
                    </main>
  • Digitando na tag Para abreviar o comando e já digitar o texto basta abrir chave na tag que deseja inserir o texto. Exemplo: 
    • 
                    main>div>p{texto texto texto}

    ficaria da segunite forma:

    
                    <main>
                        <div>
                            <p>texto texto texto</p>
                        </div>
                    </main>
  • Adicionando tag de mesmo nível semântico: para isso basta usar o sinal de + invés do >. Exemplo
    • 
                    div>h1{título título}+p{texto texto texto}

    ficaria da segunite forma:

    
                    <div>
                        <h1>título título</h1>
                        <p>texto texto texto</p>
                    </div>
  • Adicionar class: Para isso basta usar "." e o nome da class após a tag que deseja aplicar a class. Exemplo:
    • 
                    div>h1{título título}+p.mark{texto texto texto}

    ficaria da segunite forma:

    
                    <div>
                        <h1>título título</h1>
                        <p class="mark">texto texto texto</p>
                    </div>
  • Adicionar id: Igual a class porém invés de ponto bota "#". Exemplo:
    • 
                    div>h1#capitulo{título título}+p{texto texto texto}

    ficaria da segunite forma:

    
                    <div>
                        <h1 id="capitulo">título título</h1>
                        <p>texto texto texto</p>
                    </div>
  • Lista: è possível criar vários itens da mesma tag usando "*" seguido do número de vezes que desja repetir. Exemplo:
    •                     div>ul>li*5

    ficaria da segunite forma:

    
                    <div>
                        <ul>
                            <li></li>
                            <li></li>
                            <li></li>
                            <li></li>
                            <li></li>
                        </ul>
                    </div>
  • Multiplicar com subtag: Caso queira criar uma varias tag com sub tag basta multiplicar a tag e acrescentar sub tag a ela. Exemplo:
    •                     div>ul>li*5>strong

    ficaria da segunite forma:

    
                    <div>
                        <ul>
                            <li><strong></strong></li>
                            <li><strong></strong></li>
                            <li><strong></strong></li>
                            <li><strong></strong></li>
                            <li><strong></strong></li>
                        </ul>
                    </div>
  • Criando várias id É possível ainda criar várias id usando o "$" para que o vs code entenda que está enumerando, é como arrastar no excel. Exemplo: 
    • 
                    ul>li#item$*5>strong

    ficaria da segunite forma:

    
                    <ul>
                        <li id="item1"><strong></strong></li>
                        <li id="item2"><strong></strong></li>
                        <li id="item3"><strong></strong></li>
                        <li id="item4"><strong></strong></li>
                        <li id="item5"><strong></strong></li>
                    </ul>
  • Comentário: Para adicionar comentários basta usar |c. Porém isso é usado somente para identificar onde terminar as tag com class ou id. Isso serve para não confundir as finalizações de tag quando a página é grande. Exemplo:
    • 
                    section#secao$*2>article.artigo*3>h1{Título}+p{Argumentos}*5|c

    ficaria da segunite forma:

    
                    <section id="secao1">
                        <article class="artigo">
                            <h1>Título</h1>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                        </article>
                        <!-- /.artigo -->
                        <article class="artigo">
                            <h1>Título</h1>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                        </article>
                        <!-- /.artigo -->
                        <article class="artigo">
                            <h1>Título</h1>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                        </article>
                        <!-- /.artigo -->
                    </section>
                    <!-- /#secao1 -->
                    <section id="secao2">
                        <article class="artigo">
                            <h1>Título</h1>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                        </article>
                        <!-- /.artigo -->
                        <article class="artigo">
                            <h1>Título</h1>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                        </article>
                        <!-- /.artigo -->
                        <article class="artigo">
                            <h1>Título</h1>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                            <p>Argumentos</p>
                        </article>
                        <!-- /.artigo -->
                    </section>
                    <!-- /#secao2 -->
  • Tag com subtag de mesmo nível: Explicar é complicado mas imagina o seguinte exemplo onde você precisa criar um header com um h1, um main com h1 e p e um footer com h2 e p. Nesse caso as tag header main e footer são de mesmo nível e se usar o + não vai dar para usar subtag. Nesse caso precisamos usar os () parenteses. É como uma expressão matemática onde o que está entre parenteses é feito primeiro. Exemplo:
    • 
(header>h1{Capítulo})+(main>h2{Matéria}+p{lorem})+footer>p{rodapé}

    ficaria da segunite forma:

    
<header>
<h1>Capítulo</h1>
</header>

<main>
<h2>Matéria</h2>
<p>lorem</p>
</main>

<footer>
<p>rodapé</p>
</footer>
  • Adição de tag com atributos: Para adicionar atributos em tags basta colocar os atributos entre colchetes []Exemplo:
    • 
table>(thead>th[scope="col"]*3)+(tbody>tr*3>th[scope="row"]+td*2)

    ficaria da segunite forma:

        
    <table>
        <thead>
            <th scope="col"></th>
            <th scope="col"></th>
            <th scope="col"></th>
        </thead>
        <tbody>
            <tr>
                <th scope="row"></th>
                <td></td>
                <td></td>
            </tr>
            <tr>
                <th scope="row"></th>
                <td></td>
                <td></td>
            </tr>
            <tr>
                <th scope="row"></th>
                <td></td>
                <td></td>
            </tr>
        </tbody>
    </table>
