Grid-Layout

Grid layout é uma matéria relativamente simples que tem por finalidade definir o layout principal da página através de grades que se ajustam de forma responsiva. Diferentemente das flex-box que se comportam como fluido, a grid trabalha de forma amarrada, parecido com um tabela.

Conteiner

Assim como as flex-box quando trabalhamos com grid também temos uma box conteiner que abriga todas as demais box

Para isso usamos o seguintes comandos na box conteiner:

  • display: grid; Esse comando é que define que essa box é um conteiner que se comportará como grid. Esse comando é obrigatório para aplicar o grid layout
  • grid-template-columns: auto auto auto; Esse comnado define a quantidade de colunas que esse conteiner terá. Nesse caso foi definido como 3 colunas de tamanho adaptável de acordo com a tela. Entretanto, nesse caso se limita ao contúdo da box interna, quer dizer que quando não for possível reduzir mais seu conteúdo, ela irá gerar um overflow.

    • Esse comando parece um shothand, pois cada auto ou tamanho definido ( ex.: 150px auto auto) que digitar irá criará automanticamente uma nova coluna

    Por padrão ele o conteiner vem definido para uma coluna em auto

  • grid-template-rows: 150px; Semelhantes ao comando de cima só que esse funciona com colunas.Nesse caso como foi colocado apenas um 150px apenas a primeira linha terá 150px as demais ficaram como auto.

    • A diferença é que esse por padrão assume quantas linhas forem necessária para a quantidade de box internas que existirem no conteiner.

    Caso defina quatro linhas de 150px e duas colunas auto, nesse caso uma quinta linha seria criada abrigando duas box com tamanho auto.

  • column-gap: Essa propriedade define um espaço padrão para ficar entre as colunas de box internas. Ela funciona como a justify-content: space-between da flex box
  • row-gap: Essa define um espaçamento padrão entre as linha da grid. Ela funciona como a align-content: space-between; da flex-box
  • gap Essa é como uma shothand onde podemos definir os dois espaçamentos acima em uma só linha, sendo a primeira referente a linha e o segundo referente a coluna.

    • Vale ressaltar que caso defina apenas um valor, esse valor será aplicado aos dois espaçamentos

/* column-gap: 5px ; row-gap: 5px;*/ gap: 5px 10px;

itens

Vamos começar agora a falar sobre propriedade de itens da grade. Itens entende-se as box internas.

  • grid-column-start: 1De forma define em qual coluna esse item irá começar. Use o html testes para melhor compreensão.
  • grid-column-end: 2; Define até qual coluna esse item irá se esticar.Porém ele não inclui a coluna ele vai apenas até. Nessa caso como está até a coluna dois, mudaria absolutamente nada. Mas se fosse até a 3 o item esticaria absorvendo o espaço da coluna 2
  • grid-column: 1 / span 3; Esse comando é como uma shothand dos 2 citados anteriormente onde o primeiro número refere-se a coluna onde o item começa e o sedundo onde ele acaba.

    • Note de nesse caso foi adicionado um span antes do segundo número. Esse span serve apenas para incluir a coluna então nesse caso ele absorve a coluna 3.

    Importante ainda ressaltar que esse span só pode ser usados em coluna, não funciona em rows como veremos mais a frente.

    Ressalto ainda que esse span pode ser aplicado em grid-column-end também e que não é brigatório em grid-column, foi só um exemplo.
  • grid-row-start: 1 Idêntico ao grid-column-start: só que nesse caso trata-se de linha
  • grid-row-end: 1 Idêntico ao grid-column-end: só que nesse caso trata-se de linha
  • grid-row: 1/3; Idêntico ao grid-column: só que nesse caso trata-se de linha e não aceita o span
  • grid-area: 1/1/2/2;Difícil pra caralho PQP. Nesse caso o 1º item permaneceria no lugar inalterado, mas se aplicar essa configuração em qualquer outro item ele tomará o lugar do primeiro item. Mas isso é uma espécie de shothand. A ordem dos itens são:
    1. grid-row-start
    2. grid-column-start
    3. grid-row-end
    4. grid-column-end

Grid-template-areas

Esse é o sonho do criador de sites. Enfim algo simples e prático para criar layout de paginas

Essa propriedade funciona da seguinte forma:

Definimos a pripriedade grid-template-area com o seguinte comandos

No body: 

    
    <div id="conteiner">
        <div id="cabeçalho">Cabeçalho</div>
        <div id="principal">Principal</div>
        <div id="rodape">Rodapé</div>
        <div id="lateral">Lateral</div>
    </div>

No head:


div#conteiner{
    border: 2px solid #000000;
    border-radius: 5px;
    background-color: lightcoral;

    display: grid;
    grid-template-columns: auto auto auto auto;
    grid-template-rows: 150px 150px 150px 150px;
    grid-template-areas: 
    "header header header header"
    "nav main main main"
    "nav main main main"
    "footer footer footer footer"
    ;


}
div#cabeçalho{
    border: 2px solid #000000;
    border-radius: 5px;
    background-color: aqua;

    
    grid-area: header;
    

}

div#principal{
    border: 2px solid #000000;
    border-radius: 5px;
    background-color: aqua;

    grid-area: main;

}
div#rodape{
    border: 2px solid #000000;
    border-radius: 5px;
    background-color: aqua;

    grid-area: footer;


}
div#lateral{
    border: 2px solid #000000;
    border-radius: 5px;
    background-color: aqua;

    grid-area: nav;

}
