Media Queries

Media Query é a capacidade do site de alterna entre diferenetes folhas de estilo dependendo do tipo de dispositivo com as Media types que está visualizado, e de algumas outras caractiriscas que veremos no decorrer do capítulo. Pelo que entendi é como se fosse uma picture e source media das folhas de estilos.

Como já vimos anteriormente, as folhas de estilos são criadas a em arquivo CSS e são linkadas no head com o comando: 


<link rel="stylesheet" href="../../modulo-01/capitulo-12/style.css">

Porém quando usamos media query, essas folhas são usadas de acordo com a tela ou modo saída da página. Veja algumas configurações disponíveis abaixo:

Existem as seguintes configurações de folha de estilo:

  • media="print": Modo impressora. Desse modo podemos ocultar alguns menus e outros itens da página desnecessários para impressão. Veja o comando abaixo:
    • <link rel="stylesheet" href="estilos/impressora.css" media="print">
  • media="screen": Modo Tela. Esse modo se divide em outros. Mas veremos isso mais a frente. No geral serve para monitores de computadores, tablets e smartphones. Veja o código do screen abaixo:
    • <link rel="stylesheet" href="estilos/tela.css" media="screen">
  • media="all" Configuração geral que se aplica a todos os didpositivos. Hierarquicamente é como se o all fosse a folha de CSS e o screen fosse a style feita no head do html
  • media="handheld": Modo dispositivos de mão

O media screen ainda tem mais 2 adicionais que são os formatos de tela se são modo paisagem (como um notebook) ou modo retrato, como um celular ou tablet (nesse caso as tela também deitam mas gralmente a pessoas consomem conteúdo com a tela em pé mesmo)

Para cada um desses dois formatos de tela nós iremos usar um arquivo de CSS. E para referenciarmos cada um informando qual usar de acordo com o foramato da tela do usuário usaremos os seguintes comandos:

  • Modo retrato: Veja o código abaixo
    • <link rel="stylesheet" href="estilos/retrato.css" media="screen and (orientaion: portrait)">
  • Modo paisagem: Veja o código abaixo
    • <link rel="stylesheet" href="estilos/paisagem.css" media="screen and (orientation: landscape)">

Modo tudo em uma folha de css

Além dessa forma de criar várias folhas de de css temos ums opção de fazer tudo em apenas uma folha ou até mesmo no head do html usando style.

Para isso basta criar a regra (@) em style escrevendo tudo de media para frente de declaração que fazemos para link as folhas separadas. WHAT'S? . Veja o exemplo abaixo: 


<link rel="stylesheet" href="estilos/paisagem.css" media="screen and (orientation: landscape)">

Agora basta por @ seguido dessa parte marcada e em uma folha de estilo única ou em style no head do html.

Mas qual opção é melhor?

Tudo vai depender do tamanho do projeto. Basicamente quanto maior o projeto maior a folha então nessa caso é melhor deixar separado para não ficar muito bagunçado. Já projetos menores dá pra fazer até no próprio head.
