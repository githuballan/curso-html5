Animação CSS

@keyframes

A propriedade @keyframes é utilizada para criar animações em CSS. Com ela, você pode definir mudanças graduais nos estilos de um elemento ao longo do tempo.

  • from: Define o estado inicial da animação.
  • to: Define o estado final da animação.
  • 0% - 100%: Você pode usar porcentagens para definir pontos intermediários na animação. Por exemplo, 0% é o início e 100% é o final.

Propriedade animation

A propriedade animation é uma forma abreviada de definir várias propriedades de animação ao mesmo tempo. Ela permite que você controle a reprodução de animações CSS.

  • animation-name: Define o nome da animação, que é especificado usando a regra @keyframes. Exemplo: animation-name: exemplo;
  • animation-duration: Define a duração da animação. Exemplo: animation-duration: 2s;
  • animation-timing-function: Define a curva de velocidade da animação. Exemplo: animation-timing-function: ease-in-out;
    • linear: A animação tem uma velocidade constante do início ao fim.
    • ease: Este é o valor padrão. Ele cria uma transição suave que é ideal para a maioria das animações, proporcionando uma sensação natural de aceleração e desaceleração..A animação começa devagar, acelera no meio e desacelera no final.
    • ease-in-out: A animação começa devagar, acelera no meio e desacelera no final.mas de forma mais pronunciada do que ease.
    • ease-in: A animação começa devagar e acelera no final.
    • ease-out: A animação começa rápida e desacelera no final.Este valor é usado quando se deseja uma animação que tenha uma aceleração e desaceleração mais perceptíveis. É útil para criar um efeito mais dramático e enfatizar o início e o fim da animação.
    • step-start: A animação salta imediatamente para o estado final no início.
    • step-end: A animação permanece no estado inicial até o final, quando salta para o estado final.
    • steps(n, start|end): A animação é dividida em n etapas distintas. Pode começar ou terminar em um estado intermediário.
    • cubic-bezier(x1, y1, x2, y2): Permite definir uma curva de velocidade personalizada usando uma função cúbica de Bézier.
  • animation-delay: Define um atraso antes do início da animação. Exemplo: animation-delay: 1s;. Porém, esse pode ser um valor negativo, o que faz com que a animação comece como se já estivesse em andamento por essa quantidade de tempo. Exemplo: em uma animação de 4s com um delay de -2s, a página já inicia na metade da animação.
  • animation-iteration-count: Define o número de vezes que a animação deve ser reproduzida. Exemplo: animation-iteration-count: infinite;
    • n: A animação será reproduzida n vezes, onde n é um número inteiro positivo.
    • infinite: A animação será reproduzida indefinidamente.
  • animation-direction: Define se a animação deve ser reproduzida para frente, para trás ou alternadamente. Exemplo: animation-direction: alternate;
  • animation-fill-mode: Define o estilo aplicado ao elemento quando a animação não está sendo reproduzida (antes de começar, após terminar ou ambos). Exemplo: animation-fill-mode: forwards;
  • animation-play-state: Define se a animação está em execução ou pausada. Exemplo: animation-play-state: paused;
