Objeto padrão (window)

O window é o objeto que carrega todos os métodos (funções) nativos do JS, como: o console.log(), getElementById, document.innertHtml dentre muitos outros que veremos mais para frente.

por se tratar de um objeto padrão, não é necessário chamar o método da forma que chamamos os demais ecrevendo o nome do objeto e depois o método: window.console.log(), nesse caso basta escrever diretamante um de seus métodos: console.log(). Mas os dois exemplos citados funcionam.

Métodos de window

Para conhecer todos os métodos de window basta:

  1. Abrir o navegadores
  2. Botão direito
  3. Inspecionar
  4. Console
  5. Digitar window enter

Ao clicar na seta ao lado de window aparecerá todos os métodos disponíveis nesse objeto.

Para editar elementos do html usaremo os mesmos comandos vistos acima, porém serão colocados em uma variável e serão utilizados seguidos de três diferentes formas de alteração do HTML. Essas são:

Modificando Atributos

Para modificar atributos de um elemento, podemos usar as seguintes 3 propriedades:

  • setAttribute(): Para definir o valor de um atributo no elemento especificado. Se o atributo já existir, o valor será atualizado; caso contrário, um novo atributo será adicionado com o nome e valor especificados.

    Sintaxe: setAttribute(name, value)

    Outra forma de modificar ou substituir um atributo é utilizando um . (ponto)

    Sintaxe: tag.name="value"

    Caso o value tenha um "-" (traço) nessa sintaxe substituímos o - pela letra maiúscula, exemplo:

    Invés de:
    style="background-color: blue;"

    escrevemos:
    style="backgroundColor: blue;"

  • getAttribute(): Para obter o valor atual de um atributo

    Sintaxe: getAttribute(name, value)

  • removeAttribute(): Para remover um atributo

    Sintaxe: removeAttribute(name, value)

Evento JS (DOM)

Eventos com JavaScript

Os eventos permitem que o JavaScript capture interações do usuário e execute ações em resposta a essas interações. Alguns exemplos de eventos são cliques de mouse, pressionamento de teclas e movimentos do mouse.

  • onclick: Disparado quando um elemento é clicado.
  • onmouseover: Disparado quando o ponteiro do mouse passa sobre um elemento.
  • onmouseout: Disparado quando o ponteiro do mouse deixa um elemento.
  • onkeydown: Disparado quando uma tecla é pressionada.
  • onkeyup: Disparado quando uma tecla é liberada.
  • onload: Disparado quando um elemento é carregado.
  • onresize: Disparado quando a janela do navegador é redimensionada.
  • onsubmit: Disparado quando um formulário é enviado.

Exemplo de evento de clique:

<button onclick="alert('Botão clicado!')">Clique em mim</button>

Usando addEventListener

O método addEventListener permite adicionar vários eventos a um único elemento, sem sobrescrever os eventos existentes. Ele é mais flexível e poderoso do que usar atributos de evento HTML.

Os eventos mais utilizados com addEventListener incluem:

  • click: Disparado quando um elemento é clicado.
  • mouseover: Disparado quando o ponteiro do mouse passa sobre um elemento.
  • mouseout: Disparado quando o ponteiro do mouse deixa um elemento.
  • keydown: Disparado quando uma tecla é pressionada.
  • keyup: Disparado quando uma tecla é liberada.
  • load: Disparado quando um elemento é carregado.
  • resize: Disparado quando a janela do navegador é redimensionada.
  • submit: Disparado quando um formulário é enviado.

Exemplo de uso do addEventListener:

            document.getElementById("myButton").addEventListener("click", function() {
                alert("Botão clicado!");
            });

Acessando elementos pai e filho pelo JS

Para acessar elementos pai e filho pelo JS, utilizaremos as seguintes propriedades:

  • parentElement: Retorna o elemento pai do elemento especificado.
  • children: Retorna uma coleção de elementos filhos de um elemento.
  • firstElementChild: Retorna o primeiro elemento filho de um elemento.
  • lastElementChild: Retorna o último elemento filho de um elemento.
