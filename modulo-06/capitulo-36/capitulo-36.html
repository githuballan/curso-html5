Animações com HTML, CSS e JS

Para criar animações na web, podemos utilizar uma combinação de HTML, CSS e JavaScript. O HTML define a estrutura, o CSS estiliza os elementos e o JavaScript controla a animação.

Dentro dessa função, atualizamos a posição da small-box incrementando o valor da propriedade left até que ela atinja a borda direita da container. Canvas

O elemento <canvas> do HTML5 é uma poderosa ferramenta para desenhar gráficos e criar animações diretamente na página web. Ele permite que você desenhe formas, textos, imagens e até mesmo crie jogos interativos utilizando JavaScript.

Algumas funcionalidades do <canvas> incluem:

  • Desenho de formas geométricas (retângulos, círculos, linhas, etc.)
  • Renderização de textos com diferentes fontes e estilos
  • Manipulação de imagens (corte, redimensionamento, etc.)
  • Criação de animações complexas e interativas
  • Aplicação de efeitos visuais, como sombras e gradientes

Utilizar o <canvas> pode ser mais vantajoso do que fazer animações com HTML, CSS e JavaScript por várias razões:

  • Maior controle sobre os gráficos e animações, permitindo criar efeitos mais complexos e personalizados
  • Melhor desempenho, especialmente para animações e jogos que exigem renderização rápida e eficiente
  • Possibilidade de manipular diretamente os pixels da tela, oferecendo uma flexibilidade que não é possível com CSS
  • Capacidade de criar gráficos dinâmicos e interativos que respondem a eventos do usuário em tempo real

Desenhando uma Linha com Canvas

Para desenhar uma linha usando o elemento <canvas>, precisamos seguir alguns passos básicos. Primeiro, adicionamos o elemento <canvas> ao nosso HTML. Em seguida, utilizamos o JavaScript para acessar o contexto de desenho e desenhar a linha.

Vamos começar adicionando o elemento <canvas>:

<canvas id="meuCanvas" width="400" height="400"></canvas>
  • id: Um identificador único para o elemento <canvas>, que permite acessá-lo e manipulá-lo usando JavaScript.
  • width: Define a largura do elemento <canvas> em pixels.
  • height: Define a altura do elemento <canvas> em pixels.

Agora, vamos adicionar o código JavaScript para desenhar a linha:

<script>
    const canvas = document.getElementById('meuCanvas');
    const ctx = canvas.getContext('2d');

    // Definindo as propriedades da linha
    ctx.strokeStyle = 'blue'; // Cor da linha
    ctx.lineWidth = 2; // Largura da linha

    // Desenhando a linha
    ctx.beginPath(); // Inicia um novo caminho
    ctx.moveTo(50, 50); // Move o ponto inicial para (50, 50)
    ctx.lineTo(350, 350); // Desenha uma linha até (350, 350)
    ctx.stroke(); // Desenha o contorno da linha
</script>

Explicação dos elementos:

  • canvas.getContext('2d'): Obtém o contexto de desenho 2D do canvas, que nos permite desenhar formas e linhas.
  • ctx.strokeStyle: Define a cor da linha.
  • ctx.lineWidth: Define a largura da linha em pixels (importante estar sempre definida antes de lineTo).
  • ctx.beginPath(): Inicia um novo caminho de desenho.
  • ctx.moveTo(x, y): Move o ponto inicial do caminho para as coordenadas (x, y).
  • ctx.lineTo(x, y): Desenha uma linha até as coordenadas (x, y).
  • ctx.stroke(): Desenha o contorno da linha no canvas.

Desenhando um Retângulo com Canvas

Para desenhar um retângulo usando o elemento <canvas>, precisamos seguir alguns passos básicos. Primeiro, adicionamos o elemento <canvas> ao nosso HTML. Em seguida, utilizamos o JavaScript para acessar o contexto de desenho e desenhar o retângulo.

Exemplo 1: Retângulo Preenchido

<canvas id="canvas1" width="400" height="400"></canvas>
<script>
    const canvas1 = document.getElementById('canvas1');
    const ctx1 = canvas1.getContext('2d');

    // Definindo as propriedades do retângulo
    ctx1.fillStyle = 'green'; // Cor de preenchimento

    // Desenhando o retângulo preenchido
    ctx1.fillRect(50, 50, 200, 100); // Desenha um retângulo preenchido
</script>

Explicação dos elementos:

  • canvas.getContext('2d'): Obtém o contexto de desenho 2D do canvas, que nos permite desenhar formas e linhas.
  • ctx.fillStyle: Define a cor de preenchimento do retângulo.
  • ctx.fillRect(x, y, width, height): Desenha um retângulo preenchido com a cor definida em fillStyle. Os parâmetros x e y definem a posição do canto superior esquerdo do retângulo, enquanto width e height definem a largura e a altura do retângulo, respectivamente.

Exemplo 2: Retângulo com Borda de Cor Diferente

<canvas id="canvas2" width="400" height="400"></canvas>
<script>
    const canvas2 = document.getElementById('canvas2');
    const ctx2 = canvas2.getContext('2d');

    // Definindo as propriedades do retângulo
    ctx2.strokeStyle = 'blue'; // Cor da borda
    ctx2.lineWidth = 5; // Largura da borda

    // Desenhando o retângulo com borda
    ctx2.strokeRect(50, 50, 200, 100); // Desenha um retângulo com borda
</script>

Explicação dos elementos:

  • ctx.strokeStyle: Define a cor da borda do retângulo.
  • ctx.lineWidth: Define a largura da borda do retângulo.
  • ctx.strokeRect(x, y, width, height): Desenha um retângulo com borda usando a cor definida em strokeStyle e a largura definida em lineWidth. Os parâmetros x, y, width e height têm a mesma função que em fillRect.

Exemplo 3: Retângulo com "Buraco" no Centro

<canvas id="canvas3" width="400" height="400"></canvas>
<script>
    const canvas3 = document.getElementById('canvas3');
    const ctx3 = canvas3.getContext('2d');

    // Desenhando o retângulo preenchido
    ctx3.fillStyle = 'red';
    ctx3.fillRect(50, 50, 300, 200);

    // Criando um "buraco" no centro
    ctx3.clearRect(150, 100, 100, 100); // Limpa um retângulo no centro
</script>

Explicação dos elementos:

  • ctx.clearRect(x, y, width, height): Limpa a área definida pelos parâmetros x, y, width e height, tornando-a transparente. Isso cria um "buraco" no retângulo preenchido.

Desenhando Duas Linhas com Canvas

Para desenhar duas linhas usando o elemento <canvas>, precisamos seguir alguns passos básicos. Primeiro, adicionamos o elemento <canvas> ao nosso HTML. Em seguida, utilizamos o JavaScript para acessar o contexto de desenho e desenhar as linhas.

Exemplo: Duas Linhas com Canvas

<canvas id="canvas4" width="400" height="400"></canvas>
<script>
    const canvas4 = document.getElementById('canvas4');
    const ctx4 = canvas4.getContext('2d');

    // Desenhando a primeira linha
    ctx4.beginPath(); // Inicia um novo caminho
    ctx4.moveTo(50, 50); // Move o ponto inicial para (50, 50)
    ctx4.lineTo(350, 50); // Desenha uma linha até (350, 50)
    ctx4.strokeStyle = 'blue'; // Define a cor da linha
    ctx4.lineWidth = 2; // Define a largura da linha
    ctx4.stroke(); // Desenha o contorno da linha

    // Desenhando a segunda linha
    ctx4.beginPath(); // Inicia um novo caminho
    ctx4.moveTo(50, 100); // Move o ponto inicial para (50, 100)
    ctx4.lineTo(350, 100); // Desenha uma linha até (350, 100)
    ctx4.strokeStyle = 'red'; // Define a cor da linha
    ctx4.lineWidth = 2; // Define a largura da linha
    ctx4.stroke(); // Desenha o contorno da linha
</script>

Explicação dos elementos:

  • canvas.getContext('2d'): Obtém o contexto de desenho 2D do canvas, que nos permite desenhar formas e linhas.
  • ctx.beginPath(): Inicia um novo caminho de desenho. Isso é necessário para desenhar formas ou linhas separadas.
  • ctx.moveTo(x, y): Move o ponto inicial do caminho para as coordenadas (x, y).
  • ctx.lineTo(x, y): Desenha uma linha até as coordenadas (x, y).
  • ctx.strokeStyle: Define a cor da linha.
  • ctx.lineWidth: Define a largura da linha em pixels.
  • ctx.stroke(): Desenha o contorno da linha no canvas.

Se não usarmos o beginPath() ao desenhar as duas linhas, elas serão conectadas automaticamente. Isso ocorre porque o contexto de desenho do canvas mantém um caminho atual, e qualquer nova linha desenhada será adicionada a esse caminho. No exemplo acima, se omitirmos o beginPath() antes de desenhar a segunda linha, o resultado será uma linha contínua que vai de (50, 50) a (350, 50) e depois de (350, 50) a (350, 100). Para evitar isso e garantir que as linhas sejam desenhadas separadamente, é importante usar beginPath() para iniciar um novo caminho de desenho.

Exemplo: Usando o Comando closePath

O método closePath() do contexto de desenho 2D do canvas é utilizado para fechar um caminho, conectando o ponto atual ao ponto inicial do caminho. Isso é útil para desenhar formas fechadas, como polígonos.

<canvas id="canvas7" width="400" height="400"></canvas>
<script>
    const canvas7 = document.getElementById('canvas7');
    const ctx7 = canvas7.getContext('2d');

    // Desenhando um triângulo
    ctx7.beginPath(); // Inicia um novo caminho
    ctx7.moveTo(100, 100); // Move o ponto inicial para (100, 100)
    ctx7.lineTo(300, 100); // Desenha uma linha até (300, 100)
    ctx7.lineTo(200, 300); // Desenha uma linha até (200, 300)
    ctx7.closePath(); // Fecha o caminho conectando o ponto atual ao ponto inicial
    ctx7.strokeStyle = 'green'; // Define a cor da linha
    ctx7.lineWidth = 2; // Define a largura da linha
    ctx7.stroke(); // Desenha o contorno do triângulo
</script>

Explicação dos elementos:

  • ctx.closePath(): Fecha o caminho atual conectando o ponto atual ao ponto inicial do caminho.

O elemento <canvas> também permite desenhar círculos e arcos, o que é útil para criar gráficos, diagramas e outros elementos visuais. Vamos explorar como desenhar um círculo simples e um arco utilizando o contexto de desenho 2D do canvas.

Mas o que são Radianos?

Radianos são uma unidade de medida para ângulos, utilizada principalmente em matemática e física. Um radiano é definido como o ângulo formado no centro de um círculo por um arco cuja extensão é igual ao raio do círculo. Em outras palavras, um radiano é o ângulo correspondente a um arco de comprimento igual ao raio do círculo. Existem 2π radianos em uma circunferência completa, o que equivale a 360 graus. Portanto, para converter graus em radianos, multiplicamos o valor em graus por π/180.

Exemplo: Desenhando um Arco com Canvas usando Radianos

    const ctx6 = canvas6.getContext('2d');

    ctx6.strokeStyle = 'orange'; // Define a cor da linha
    ctx6.lineWidth = 3; // Define a largura da linha
    ctx6.stroke(); // Desenha o contorno do arco

Explicação dos elementos:

  • ctx.arc(x, y, radius, startAngle, endAngle): Desenha um arco ou círculo. Os parâmetros x e y definem o centro do arco, radius define o raio, startAngle e endAngle definem o ângulo inicial e final do arco em radianos.

Exemplo: Animação com Canvas usando setInterval

Além de requestAnimationFrame, também podemos usar setInterval para criar animações no canvas. Vamos explorar como criar uma animação simples utilizando o contexto de desenho 2D do canvas e a função setInterval.

<canvas id="canvasAnimacaoInterval" width="400" height="400"></canvas>
<script>
    const canvasAnimacaoInterval = document.getElementById('canvasAnimacaoInterval');
    const ctxAnimacaoInterval = canvasAnimacaoInterval.getContext('2d');
    let xInterval = 0;
    let yInterval = 200;
    let dxInterval = 2;

    function drawInterval() {
        ctxAnimacaoInterval.clearRect(0, 0, canvasAnimacaoInterval.width, canvasAnimacaoInterval.height);
        ctxAnimacaoInterval.beginPath();
        ctxAnimacaoInterval.arc(xInterval, yInterval, 20, 0, 2 * Math.PI);
        ctxAnimacaoInterval.fillStyle = 'red';
        ctxAnimacaoInterval.fill();
        ctxAnimacaoInterval.stroke();
        xInterval += dxInterval;
        if (xInterval > canvasAnimacaoInterval.width - 20 || xInterval < 20) {
            dxInterval = -dxInterval;
        }
    }

    setInterval(drawInterval, 20);
</script>

Explicação dos elementos:

  • setInterval(callback, delay): Chama repetidamente a função callback a cada delay milissegundos.
  • ctx.clearRect(x, y, width, height): Limpa a área definida pelos parâmetros x, y, width e height, permitindo que a animação seja redesenhada.
  • ctx.arc(x, y, radius, startAngle, endAngle): Desenha um arco ou círculo. Os parâmetros x e y definem o centro do círculo, radius define o raio, startAngle e endAngle definem o ângulo inicial e final do arco em radianos.

Exemplo: Animação com requestAnimationFrame

O elemento <canvas> também permite criar animações, o que é útil para criar gráficos dinâmicos, jogos e outros elementos interativos. Vamos explorar como criar uma animação simples utilizando o contexto de desenho 2D do canvas.

<canvas id="canvasAnimacao" width="400" height="400"></canvas>
<script>
    const canvasAnimacao = document.getElementById('canvasAnimacao');
    const ctxAnimacao = canvasAnimacao.getContext('2d');
    let x = 0;
    let y = 200;
    let dx = 2;

    function draw() {
        ctxAnimacao.clearRect(0, 0, canvasAnimacao.width, canvasAnimacao.height);
        ctxAnimacao.beginPath();
        ctxAnimacao.arc(x, y, 20, 0, 2 * Math.PI);
        ctxAnimacao.fillStyle = 'blue';
        ctxAnimacao.fill();
        ctxAnimacao.stroke();
        x += dx;
        if (x > canvasAnimacao.width || x < 0) {
            dx = -dx;
        }
        requestAnimationFrame(draw);
    }

    draw();
</script>

Explicação dos elementos:

  • ctx.clearRect(x, y, width, height): Limpa a área definida pelos parâmetros x, y, width e height, permitindo que a animação seja redesenhada.
  • ctx.arc(x, y, radius, startAngle, endAngle): Desenha um arco ou círculo. Os parâmetros x e y definem o centro do círculo, radius define o raio, startAngle e endAngle definem o ângulo inicial e final do arco em radianos.
  • requestAnimationFrame(callback): Solicita que o navegador chame uma função específica para atualizar uma animação antes da próxima repintura.

Exemplo: Carregando Imagem no Canvas

O elemento <canvas> permite carregar e manipular imagens, o que é útil para criar gráficos dinâmicos e interativos. Vamos explorar como carregar uma imagem no canvas utilizando JavaScript e como usar as propriedades naturalWidth e naturalHeight para obter as dimensões naturais da imagem.

<canvas id="canvasImagem" width="400" height="400"></canvas>
<script>
    const canvasImagem = document.getElementById('canvasImagem');
    const ctxImagem = canvasImagem.getContext('2d');
    const img = new Image();
    img.src = './pedra.jpeg';

    img.onload = function() {
        const imgWidth = img.naturalWidth;
        const imgHeight = img.naturalHeight;
        console.log('Largura natural:', imgWidth);
        console.log('Altura natural:', imgHeight);
        ctxImagem.drawImage(img, 0, 0, canvasImagem.width, canvasImagem.height);
    };
</script>

Explicação dos elementos:

  • img.src: Define o caminho da imagem a ser carregada.
  • img.onload: Função que é chamada quando a imagem é carregada. Dentro dessa função, podemos manipular a imagem no canvas.
  • img.naturalWidth: Obtém a largura natural da imagem.
  • img.naturalHeight: Obtém a altura natural da imagem.
  • ctx.drawImage(img, x, y, width, height): Desenha a imagem no canvas. Os parâmetros x e y definem a posição no canvas, e width e height definem as dimensões da imagem desenhada.
