Biblioteca Math

A biblioteca Math é uma biblioteca padrão do JavaScript que fornece funções matemáticas básicas e avançadas. Ela é usada para realizar operações matemáticas como arredondamento, geração de números aleatórios, trigonometria, entre outras.

Aqui estão alguns dos métodos mais usados da biblioteca Math:

  • Math.random(): Gera um número pseudo-aleatório entre 0 (inclusive) e 1 (exclusivo). 
    let randomNum = Math.random();
  • Math.round(): Arredonda um número para o inteiro mais próximo. 
    let roundedNum = Math.round(4.7); // 5
  • Math.floor(): Arredonda um número para baixo, para o inteiro mais próximo. 
    let flooredNum = Math.floor(4.7); // 4
  • Math.ceil(): Arredonda um número para cima, para o inteiro mais próximo. 
    let ceiledNum = Math.ceil(4.3); // 5
  • Math.max(): Retorna o maior número de uma lista de argumentos. 
    let maxNum = Math.max(1, 2, 3, 4, 5); // 5
  • Math.min(): Retorna o menor número de uma lista de argumentos. 
    let minNum = Math.min(1, 2, 3, 4, 5); // 1
  • Math.sqrt(): Calcula a raiz quadrada de um número. 
    let sqrtNum = Math.sqrt(16); // 4
  • Math.pow(): Calcula a potência de um número. 
    let powNum = Math.pow(2, 3); // 8

Esses são apenas alguns exemplos das muitas funções disponíveis na biblioteca Math. Ela é uma ferramenta poderosa para qualquer tipo de cálculo matemático em JavaScript.

JSON- Java Script Object Notation

JSON (JavaScript Object Notation) é um formato leve de intercâmbio de dados. Ele é fácil para os humanos lerem e escreverem, e fácil para máquinas interpretarem e gerarem. JSON é uma representação textual de um objeto JavaScript, mas é independente de linguagem, o que significa que pode ser usado em muitas linguagens de programação.

Um objeto JavaScript é uma coleção de pares chave-valor, onde as chaves são strings e os valores podem ser qualquer tipo de dado, incluindo outros objetos. JSON segue a mesma estrutura, mas é representado como uma string.

Métodos

  • JSON.parse() é usado para converter uma string JSON em um objeto JavaScript.
  • JSON.stringify() é usado para converter um objeto JavaScript em uma string JSON.

Aqui está um exemplo que mostra a diferença entre os dois métodos:

// Objeto JavaScript
let obj = {
    nome: "Allan",
    idade: 30,
    cidade: "São Paulo"
};

// Convertendo o objeto JavaScript em uma string JSON
let jsonString = JSON.stringify(obj);
console.log(jsonString); // {"nome":"Allan","idade":30,"cidade":"São Paulo"}

// Convertendo a string JSON de volta para um objeto JavaScript
let newObj = JSON.parse(jsonString);
console.log(newObj); // {nome: "Allan", idade: 30, cidade: "São Paulo"}

Devemos usar JSON em vez de objetos JavaScript quando precisamos enviar dados entre um servidor e um cliente. JSON é um formato de dados leve e independente de linguagem, o que o torna ideal para a troca de dados em aplicações web.

Local Storage

O armazenamento local (local storage) é uma maneira de armazenar dados no navegador do usuário. Ele permite que você armazene dados no navegador do usuário de forma persistente, o que significa que os dados permanecem mesmo depois que o navegador é fechado.

O armazenamento local é útil para armazenar informações como configurações de usuário, preferências, dados de formulários, entre outros. Ele é uma alternativa ao uso de cookies, que são limitados em tamanho e têm outras limitações.

Métodos

  • localStorage.setItem() é usado para armazenar um par chave-valor no armazenamento local.
  • localStorage.getItem() é usado para recuperar o valor associado a uma chave no armazenamento local.
  • localStorage.removeItem() é usado para remover um item do armazenamento local.
  • localStorage.clear() é usado para remover todos os itens do armazenamento local.

Aqui está um exemplo que mostra como usar o armazenamento local:

// Armazenando um valor no armazenamento local
localStorage.setItem("nome", "Allan");

// Recuperando o valor do armazenamento local
let nome = localStorage.getItem("nome");
console.log(nome); // Allan

// Removendo o valor do armazenamento local
localStorage.removeItem("nome");

// Limpando o armazenamento local
localStorage.clear();

O armazenamento local é uma ferramenta poderosa para armazenar dados no navegador do usuário. Ele é amplamente suportado pelos navegadores modernos e é fácil de usar. No entanto, é importante lembrar que o armazenamento local é limitado em tamanho (geralmente 5 MB) e não deve ser usado para armazenar grandes quantidades de dados.
