Introdução ao CSS3

Estilo CSS3 inline

O estilo inline é utilizado para modificar o estilo de elementos específicos diretamente no HTML. Esse estilo é usado em casos raros onde você precisa de uma modificação muito pontual, como destacar um trecho específico de um texto.

Exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Nam veniam rem, qui rerum corrupti nobis dignissimos incidunt, quos nesciunt minima iure maxime alias? Nihil accusamus cupiditate ad commodi velit numquam!

Estilo CSS3 Interno

O estilo CSS3 interno é definido dentro da tag <style> no cabeçalho da página. Esse estilo é aplicado a toda a página, permitindo que você defina estilos para múltiplos elementos de uma só vez.

Por exemplo, todos os elementos <h1> terão o mesmo estilo, todos os elementos <h2> terão outro estilo, e assim por diante.

OBS.: Embora seja possível definir estilos no corpo da página, isso não é recomendado, pois o navegador carrega primeiro o cabeçalho. Definir estilos no corpo pode causar um efeito visual desagradável, onde a página aparece sem estilo antes de ser completamente carregada.

Veja um exemplo prático neste link.

Estilo CSS3 Externo

O estilo CSS3 externo é definido em um arquivo separado com extensão .css. Esse arquivo é então vinculado ao HTML através da tag <link> no cabeçalho da página.

Trabalhar com CSS externo é vantajoso porque permite que você aplique o mesmo estilo a várias páginas do site. Se precisar alterar a formatação, basta modificar o arquivo CSS, e todas as páginas vinculadas a ele serão atualizadas automaticamente.

Criando o Link CSS

No cabeçalho da página, adicione a seguinte linha de código:

                    <link rel="stylesheet" href="style.css">

Para criar um arquivo CSS no VS Code, navegue até a pasta de destino, clique com o botão direito e selecione "Novo Arquivo". Nomeie o arquivo com a extensão .css.

Abra o arquivo CSS e adicione a seguinte linha no início:

@charset "UTF-8";

Por que usar @charset "UTF-8"?

A declaração @charset "UTF-8"; é usada para definir a codificação de caracteres do arquivo CSS. Isso é importante porque garante que todos os caracteres, incluindo acentos e caracteres especiais, sejam exibidos corretamente em diferentes navegadores e sistemas operacionais.

Sem essa declaração, pode haver problemas de exibição, especialmente em idiomas que utilizam caracteres acentuados ou símbolos específicos. Definir a codificação como UTF-8 é uma boa prática para garantir a compatibilidade e a correta renderização do conteúdo.

Em seguida, defina os estilos desejados. Exemplo:

    body {
        background-color: rgba(0, 255, 135, 0.03);
        font-size: large;
        margin: 5% 5% 10% 5%;
    }

    h1 {
        background-color: rgba(0, 255, 255, 0.2);
        font-size: 25px;
        text-align: center;
    }

    h2 {
        background-color: rgba(0, 235, 255, 0.1);
        font-size: 20px;
    }

Hierarquia de Estilos CSS3

Existem três formas de aplicar estilos ao site: inline, interno e externo. É possível usar os três simultaneamente, e por isso existe uma hierarquia entre eles. O HTML prioriza do mais específico para o mais generalizado.

Ordem de prioridade:

  1. Estilo Inline
  2. Estilo CSS3 Interno
  3. Estilo CSS3 Externo

OBSERVAÇÃO: Se houver estilos concorrentes na mesma folha de estilo e de mesma hierarquia, o CSS aplicará o que for definido por último.
