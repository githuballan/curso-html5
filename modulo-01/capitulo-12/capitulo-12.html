Iniciando CSS3

Estilo InLine

O estilo inline é utilizado para modificar o estilo linha por linha, esse estilo é utilizado é raros casos onde você precisará de uma modificação muito pontual, como destacar um trecho específico de um texto.
Exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Nam veniam rem, qui rerum corrupti nobis dignissimos incidunt, quos nesciunt minima iure maxime alias? Nihil accusamus cupiditate ad commodi velit numquam!

Estilo CSS3 Interno

O estilo CSS3 interno invés de ser feito linha por linha ele é feito na parte head da página. Assim o estilo modifica a parte inteiras da página.
Parte inteiras são todos os <h1> <h1/> ficam com o mesmo determinada estilo, todo <h2> <h2/> em outro todo <p> </p> em outro e assim por diante.
O Exemplo será aplicado nesse link.

Estilo CSS3 Externo

O link externo funciona como o uso de variáveis do tipo C para identificar cada porta por função assim caso precisa-se trocar alguma fio de porta você não precisaria trocar as portas no programa inteiro, basta trocar a porta na variável.
A ideia de trabalhar com uma CSS externa é essa, caso precisa trocar alguma formatação você não precisará trocar página por página.
Basta criar o link para o arquivo CSS no head do HTML e colocar o link ,para esse arquivo, em todas as páginas do site para que fiquem sempre com a mesma formatação.

Criando o link CSS

Va em alguma área de head escreva link e selecione link:css na lista, imediatamente aparecerá o code abaixo
Exemplo:
<link rel="stylesheet" href="style.css">

Atalho:
Para criar um arquivo dentro do Code basta ir na área de destino do arquivo no código e pressional "CTRL+Click na área" assim o VS Code irá criar um arquivo na pasta onde está o arquivo atual OBS.:Deve lembrar sempre de por também a extenção do arquivo

Abra esse arquivo com o VS Code e escreva @(@ significa regra):
@charset "UTF-8";
e logo abaixo basta escrever os elementos que deseja editar:

Hierarquia de Estilos

Como vimos existem 3 formas de aplicar estilos ao site. Porém é possível usar os 3 simultaneamente e por isso existe um Hierarquia entre eles. No caso o HTML prioriza do mais específico para o mas generalizado.
Veja o exemplo abaixo na ordem de prioridade:
  1. Estilo InLine
  2. Estilo CSS3 Interno
  3. Estilo CSS3 Externo
