Formas de destacar texto em HTML5:

Negrito Para destacar em negrito de forma semântica usamos o comando <strong>"Texto a ser destacado"</strong>.
Exemplo: Negrito

Itálico Para destacar em Itálico de forma semântica usamos o comando <em>"Texto a ser destacado"</em>.
Exemplo: Itálico.

Marca texto Para destacar em usando efeitio caneta de marca texto de forma semântica usamos o comando <mark>"Texto a ser destacado"</mark>.
Exemplo: Marca texto
Uma breve passagem por CCS Caso queira mudar a cor do marca texto pode-se usar dois comandos diferentes de CCS
1º mudança de cor individual: Escreve <mark style="background-color: lime;">"Texto a ser destacado"</mark>.
Exemplo: Marca texto
2º mudança de cor de todos os marca texto da página: Na área head em baixo de title escreva: <style> mark{background-color: "Selecione uma cor na lista";} </style>

Sublinhado:Para destacar sublinhando-o de forma semântica usamos o comando <ins>"Texto a ser destacado"</ins>.
Exemplo: Sublinhado.

Tachado:Para destacar um texto tachando-o de forma semântica usamos o comando <del>"Texto a ser destacado"</del>.
Exemplo: Tachado.

Texto sobrescrito:Para escrever um texto ou algo sobrescrito usamos o comando <sup>"Texto a ser sobrescrito"</sup>.
Exemplo: Texto sobrescrito.

Texto subescrito:Para escrever um texto ou algo subescrito usamos o comando <sub>"Texto a ser subescrito"</sub>.
Exemplo: Texto subescrito.

Código fonte: Comando usado para escrever um texto com uma fonte monoespaçada,( o que isso significa? Isso é um texto onde todas a letras ocupam o mesmo espaçamento). Esse tipo de fonte é ótima para fazer leitura de códigos. Para isso usamos o comando <code>"Texto monoespaçado"</code>. A seguir veja 2 exemplos onde o primeiro está monoespaçado e o segundo está normal:

Comando pre: Esse comando faz, a organização de espaçamentos e quebra de linhas de página, ficarem iguais aos feito no Visual Studio. Para isso usamos o comando <pre>"Texto"</pre>. Exemplo:

Atalho:
Para espaçar ou reduzir o espaçamento dos parágrafos de textos inteiros no Visual Studio:
Para aumanetar o espaço: selecionar o texto inteiro e pressionar a tecla "Tab"
Para remover o espaçamento: Basta selecionar o texto inteiro e pressionar a tecla "Shift" e "Tab"

Citação Simples:Para destacar uma citação usamos as famosas aspas, porém para que o programa saiba de se trata de uma citação usamos o comando <q>"Texto citado"</q>.
Exemplo: Como dizia fulano: Blá-Blá-Blá...

Citação completa:Para destacar uma citação completa, também usamos as famosas aspas. Porém em uma citação completa ela quebra a linha e abre um espaço depois da margem e ainda podemos referenciar, ou não, o site de onde tiramos essa citação.
Para uma citação completa sem referenciar usamos o comando <blockquote>"Texto citado"</blockquote>
Exemplo sem referência: Como dizia fulano:

Blá-Blá-Blá...

Para uma citação referenciada usamos:<blockquote cite="https://rj.olx.com.br/">"Texto citado"</blockquote>
Exemplo com referência: Como dizia fulano:
Blá-Blá-Blá...

Como pôde observar o usuário da página não verá diferença alguma entre com e sem essa referência. Isso é apenas para que o programa use para fins de pesquisas do google e afins.

Abreviações:Usamos essa tag para que quando o usuário passe o mouse por cima, ele veja o significado daquela sigla. Para isso usamos o comando <abbr title="significado da sigla" >"Sigla"</abbr>.
Exemplo: HTML

Texto invertido: Usamos essa tag para abosolutamente nada, mas ela existe então está aí. Para isso usamos o comando <bdo dir="rtl" >"Texto invertido"</bdo>.
Exemplo: Texto invertido
