Capítulo 09

Listas

Exstem 2 tipos de listagens as ordenadas (como uma lista de tarefas para fazer em sequência) e as que não possuem ordenação específica (como uma lista de compra de supermercado)

Listas não ordenadas

Para criar uma lista sem ordenação específica utilizamos a tag <ul> (ul=unordenade list)seguido da tag <li> (li=list item).

Para isso usamos a tag da seguinte forma:

<ul>
<li> Café </li>
<li> Açucar</li>
<li> Cebola </li>
<li> Batata </li>
<li> Etc.. </li>
</ul>

Exemplo:

  • Café
  • Açucar
  • Cebola
  • Batata
  • Etc..

Obs.: Não é necessário fechar a tag <li> com a tag </li> ela também pode ser escrita da seguinte forma:

<ul>
<li> Café
<li> Açucar
<li> Cebola
<li> Batata
<li> Etc..
</ul>

Adicionando estilo "Type"

Temos ainda a opção de personalizar a lista trocando o ponto da não ordenada por algum outro formato abaixo:

Podemos usar 3 formatos :

<ul type="square"> Para quadrado;
<ul type="circle"> Para um círculo ou:
<ul type="disc"> essa é a padrão

Exemplo do tipo square:

  • Café
  • Açucar
  • Cebola
  • Batata
  • Etc..

Lista ordenada

Para criar uma lista ordanada em HTML, utiliza-se a tag <ol> (ol=ordenade list)seguido da tag <li> (li=list item) pois caso crie um lista muito grande e precise alterar algum item do início, você não precisará vir trocando número por número até o final.

Para isso usamos a tag da seguinte forma:

<ol>
<li> Acordar </li>
<li> Preparar o Café </li>
<li> Tomar Banho </li>
<li> Tomar o café </li>
<li> Etc.. </li>
</ol>
Exemplo:
  1. Acordar
  2. Preparar o Café
  3. Tomar Banho
  4. Tomar o café
  5. Etc..

Adicionando estilo "Type"

Temos também a opção de personalizar a lista trocando o numeral por letras ou algarismos romano, no caso da ordenada e até iniciar a lista apartir de outra ordem que não seja o 1º item.

Podemos fazer das seguintes forma: <ol type="I"> Para algarismos Romanos Maiúsculos.
<ol type="i"> Para algarismos Romanos Minúsculos.
<ol type="A"> Para ordem alfabética Maiúscula.
<ol type="a"> Para ordem alfabética Minúscula.
Exemplo do tipo algarismos Romanos Maiúsculos:
  1. Acordar
  2. Preparar o Café
  3. Tomar Banho
  4. Tomar o café
  5. Etc..

Iniciando em outra ordem "Start"

Para começar de outra ordem que não seja a primeira basta fazer da seguinte forma:

<ol type="I" start="3"> Para algarismos Romanos Maiúsculos iniciando da 3ª ordem.

Exemplo começando da ordem 3 em romanos:

  1. Acordar
  2. Preparar o Café
  3. Tomar Banho
  4. Tomar o café
  5. Etc..

Iniciando em outra ordem inversa

Para criar um lista na ordem decrescente basta adicionar o atributo reversed na tag ol

Exemplo:

  1. Acordar
  2. Preparar o Café
  3. Tomar Banho
  4. Tomar o café
  5. Etc..

O comando

            
            <ol reversed >
                <li>Acordar</li>
                <li> Preparar o Café</li>
                <li>Tomar Banho</li>
                <li>Tomar o café</li>
                <li>Etc..</li>
            </ol>

Mesclando listas

Ainda sobre listas, mais uma possibilidade interessante é mesclar listas uma dentro de outra lista. Para exeplificar vamos criar um ídice desse próprio resumo com títulos e subtítulos e subtítulos de subtítulos.

  1. Módulo
    1. Capítulo
      • Caracteres
      • Parágrafos
      • Quebras de linha
    2. Capítulo
      • Imagens
      • Favicon
    3. Capítulo
      • Semântica
      • Hierarquia de Títulos
    4. Capítulo
      • Formatação de textos

O comando fica escrito de seguinte forma:

                
<ol>
    
  • Módulo
    • 
<ol start="5">
    <li> Capítulo</li>
    <ul>
        <li>Caracteres</li>
        <li>Parágrafos</li>
        <li>Quebras de linha</li>
    </ul>
    <li> Capítulo</li>
    <ul>
        <li>Imagens</li>
        <li>Favicon</li>
    </ul>
</ol>
etc..
</ol>

    Lista de definições

    A lista de definição funciona como se fosse um dicionário, serve para escrever o que significa determinado termo. Essa lista é bem menos usada que as demais, porém ela é útil e caso alguem pesquise no Google a definição de determinado termo ele prioriza mostrar as página que usam essa lista. Para isso usamos as seguinte Tags:
    <dl> "Definition List" tag que cria a lista
    <dt> "Definition Term" tag para escrever o termo
    <dd> "Definition Definition" tag para o sgnificado.
    </dl> finaliza a lista

    Exemplo:

    Lista de definição
    Termo
    Descrição / significado do termo
    Termo
    Descrição / significado do termo
    Termo
    Descrição / significado do termo
    Fim da lista.

    Código fica da seguinte forma: 
                <dl>Lista de definição
                    <dt>Termo</dt>
                    <dd>Descrição / significado do termo </dd>
                    <dt>Termo</dt>
                    <dd>Descrição / significado do termo </dd>
                    </dl>
                    <dt>Termo</dt>
                    <dd>Descrição / significado do termo </dd>
                </dl>
                </dl>

    Exemplo:

    Lembra a usabilidade da tag de abreviação só que exposta.
    <abbr title="significado da sigla" >"Sigla"</abbr>

    Style, Um pouco de CSS

    OBS.: Essa parte está no capítulo 17 do módulo 02 no vídeo Rodpé, conteúdo periféricos e links Minuto 06:40

    Agora, vou te mostrar como criar mais personalizações e a criar um ótimo resultado visual.

    Dividindo a lista em 02 colunas

    Para dividir a lista em mais de um coluna usamos a style columns

    Exemplo:

    1. Acordar
    2. Preparar o Café
    3. Tomar Banho
    4. Tomar o café
    5. Etc..

    Exemplo de código: 

            <ol style="columns: 2;" type="I" >
            <li>Acordar</li>
            <li> Preparar o Café</li>
            <li>Tomar Banho</li>
            <li>Tomar o café</li>
            <li>Etc..</li>
        </ol>

    Estilos externo

    É possível também usar outros marcadores além dos vistos em type, para isso temos que usar o CSS também

    Para isso usaremos a style: list-style-type: 'código'

    Exemplo:

    1. Acordar
    2. Preparar o Café
    3. Tomar Banho
    4. Tomar o café
    5. Etc..
    Exemplo de código: 
            <ol style="list-style-type: '\2714'" >
                <li>Acordar</li>
                <li> Preparar o Café</li>
                <li>Tomar Banho</li>
                <li>Tomar o café</li>
                <li>Etc..</li>
            </ol>
    Caso não lembre como usar a emoji pedia reveja o Capítulo 05

    Remover o marcador

    Para remover o marcador basta escrever "none" onde ficaria o código da emoji pedia.

    Exemplo:

    1. Acordar
    2. Preparar o Café
    3. Tomar Banho
    4. Tomar o café
    5. Etc..

    Exemplo de código:

            <ol style="list-style-type: none" >
                <li>Acordar</li>
                <li> Preparar o Café</li>
                <li>Tomar Banho</li>
                <li>Tomar o café</li>
                <li>Etc..</li>
            </ol>

    Como deixar o marcador mais perto

    É possível também deixar os marcadores mais perto dos itens na lista, para isso também temos que usar o CSS também

    Para isso usaremos a style: list-style-position: inside

    Exemplo:

    1. Acordar
    2. Preparar o Café
    3. Tomar Banho
    4. Tomar o café
    5. Etc..

    Exemplo de código: 

            <ol style="list-style-position: inside" >
                <li>Acordar</li>
                <li> Preparar o Café</li>
                <li>Tomar Banho</li>
                <li>Tomar o café</li>
                <li>Etc..</li>
            </ol>

    Como espaçar o marcador

    Se não gostou, achou que ficou muito perto. Então vamos definir a ditância manualmente

    Para isso usaremos a style: list-style-type: \2714\0020

    Exemplo:

    1. Acordar
    2. Preparar o Café
    3. Tomar Banho
    4. Tomar o café
    5. Etc..

    Exemplo de código: 

            <ol style="list-style-type: '\2714\0020';list-style-position: inside" >
                <li>Acordar</li>
                <li> Preparar o Café</li>
                <li>Tomar Banho</li>
                <li>Tomar o café</li>
                <li>Etc..</li>
            </ol>

    OBS.: 0020 é um hexadecimal para espaço assim como o 00A0 também é.

    OBS 02.: Só é possível aumentar o espaço se usar um estilo externo, caso contrário ele apenas substituirá o marcador por um espaço vazio.

    Exemplo:

    1. Acordar
    2. Preparar o Café
    3. Tomar Banho
    4. Tomar o café
    5. Etc..
