Semântica

Quando cria-se um site em HTML 5(ou HTML Semântico) não se faz modificação de cores formatos de textos ou efeitos visuais utilizando o HTML, esse tipo de efeitos/embelezamento da página é feito usando o CCS que veremos posteriormente.
Mas por que disso?
Isso é feito pra ter funcionalidades e não como apenas leitura, pois se deseja destacar algo no texto, é porque aquilo tem algum significado especial logo ele pode ter uma funcionalidade uma praticidade para o usuário final. Essas mudanças do formato antigo é para que o programa também saiba do que se trata cada item.
Por exemplo: quando digitamos um endereço invés de escrever sublinhado com <u>:
rua fernandes leão, 67, vaz lobo , Rio de janeiro, RJ
Nós escrevemos como <address>:
rua fernandes leão, 67, vaz lobo , Rio de janeiro, RJ
Assim um usuário num smartphone pode simplemente clicar no endereço e já abrir no Maps e ir para o local no mapa. Por isso os títulos são escritos dentro de <h1>, invés de apenas mudar para negrito e aumanetar ou diminuir a fonte. Por isso títulos têm Hierarquia entre eles. Futuramente ao usar CCS ou Java Script poderemos usar essas funções para gerar ações mais intuitivas cliente.

Hierarquia de Títulos

A Hierarquiade títulos é a organização de títulos e subtítulos como já vimos anteriormente <h1> é o comando usado para criar título porém existe também <h2> ,<h3> ,<h4> ,<h5> e <h6> que são usados como subtítulos de subtítulos até no máximo a 6ª Hierarquia
Atalho:
Para ganhar velocidade: Quando quiser usar uma tag sobre uma frase já digitada como em negrito basta: apertar "ctrl + shift + p" digitar abb e selecionar "Wrap with abbreviantion" depois digita a Tag que desejar <b> apenas a letra "b" e "enter".
Ou se ja tiver usado esse atalho recentemente basta digitar "ctrl + shift + p" →"Enter"→ a letra da tag que deseja usar e→ "Enter".
