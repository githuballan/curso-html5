Imagens

No HTML usamos basicamente 3 tipos de imagens JPG, PNG e GIF. Cada uma tem uma característica única que a trona perfeita para cada finalidade:
JPG: A principal característica do formato JPG é seu tamanho reduzido, permitindo que a pagina carregue em alta velocidade. Vale ressaltar que a velocidade de carregamento é uma das principais característica que um site deve ter.
PNG: Esse formato tem como principal característica a capacidade da imagem ficar trasparente. Isso permite efeitos que dão um aspecto mais profissional ao site.
GIF: Esse formato é famoso, ele permite exeibição de imagens sequenciais dando animação ao site de forma leve, sem pesar no carregamento.

Outro Ponto de Suma imortância ao trabalhar com imagens é sempre reduzir o tamanho delas para o menor possível sem afetar a boa aparência dela, pois caso não faça isso o site pode ficar extremamente pesado e gerar uma experiência de carregamento lenta. tag para imagens

tag img

Para carregar imagem usa-se a tag self-close <img> e utilizamos os seguintes atributos:

  • src: Onde coloca o endereço da imagem, para isso basta aperta ctrl+espaço e selecionar a imagem. Caso esteja em pasta anterior basta digitar../ para voltar uma pasta(pode digitar ../ quantas vezes precisar até chega na pasta desejada)
  • alt: Texto alternativo, ele aparecerá caso por algum motivo a imagem não carregue. Mas pricipalmente, será lida por software de acessibilidade em caso de pessoes com deficiência visual
  • width: Define a largura da que a imagem será exbida e a altura segue a proporção.
  • height: Define a altura da que a imagem será exbida e a largura segue a proporção.

    • Caso use height e width simultaneamente você ditará as proporções

tag figure

Essa tag trabalha junto com a tag figcaption. Onde a tag figure engloba as tag img e figcaption.

A tag figcaption. Serve para descrever a imagem mas diferente das tag alt ela carrga junto com a imagem.

Veja um exemplo de código e suas aplicações abaixo


<figure>

    <img src="../../imagens/escritorio 500 jpeg.jpg" alt=" treino">
    <figcaption>Imagem de 500px de largura </figcaption>
</figure>
Atalho

Para carregar imagens 1º digita "img" e "enter" depois clica entre as primeiras aspas após "src" e aperta "ctrl" e "espaço" abrirá a pasta onde está o arquivo html que está desenvolvendo está localizado, após isso basta selecionar a imagem que será carregada.


Boas Práticas: Criar uma pasta somente para imagens na mesma pasta onde está desenvolvendo seu site e renomear as imagens de acordo com o conteúdo e aplicação delas no site. Isso ajuda a rankear a imagem quando aparecer em mecanismos de busca.

Imagem de link Externo

Outra forma de carregar imagens para o site, é usando imagens de outros sites, para isso basta ir na imagem desejada no tal site, clicar com botão direito sobre a imagem e clicar em copiar endereço da imagem. Após isso basta colar esse endereço entre as aspas em "src=" e preecher o "alt=" normalmente.

Porém esse processo cria uma dependência de um link externo, logo se o link externo ou mudar algo no endereço o site terá um ERRO no carregamento daquela imagem.

Para inseririrmos um favicon em nosso site é fácil. basta ir em head logo antes de title e digitar link selecionar a opção: "Link:favicon" depois clicar entre as aspas logo após "href=""" e executar o mesmo procedimentousado para caregar um imagem("ctrl" + espaço → selecionar a imagem tipo ICO ) e pronto.
Como criar uma imagem tipo .ICO:
Existem alguns sites próprios para baixar imagens no formato .ICO, ou criar, ou até mesmo traformar imagens de .PNG para o formato .ICO. Alguns desses sites são:
Favicon.IO: Esse site é um dos melhores pois esse dá a opção de criar um ICO apartir de uma palavra ou abreviação curta, ou criar um ICO apartir de um PNG ou até utilizar um emogi como favicon.
Favicon.cc: Nesse site você pode criar seu prórpio ICO
IconArchive: Aqui você ja encontra os ICO prontos, basta pesquisar o tema que desejar e baixar o que mais te agradar.
