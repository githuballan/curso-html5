if e else

If e else funciona como if else da linguagem C, C++, é tudo igual, só muda a forma de escrever.

É uma função de condições booleanas, funciona parecida com o ternário que vimos anteriormente, porém o if permite um nível bem superior de complexidade nas condições.

O comando if permite que valores booleanos sejam testados, funciona parecida com o ternário que vimos anteriormente. Se o valor passado como parâmetro para o comando if for true, o bloco do if é executado. Caso caso seja false ele executará o else.

Vale ressaltar que o bloco if ou else só será lido caso a condição seja satisfeita caso contrário o programa irá pular o bloco inteiro.
