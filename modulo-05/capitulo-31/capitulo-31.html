Array

Os arrays em JavaScript são estruturas de dados utilizadas para armazenar múltiplos valores em uma única variável. Eles permitem que você organize e manipule conjuntos de dados de forma eficiente, acessando cada elemento através de um índice numérico.

Os índices dos arrays começam sempre do "0" (zero) e vão até o número do último elemento do array menos um. Essa sequência é chamada de índice do array ou, em inglês, array index.

Declarando Array

Existem duas formas de declarar um array:

Usando o construtor Array

    familia = new Array("Allan", "Karol", "Maya", "Moana");
    console.log(familia[3]); // Moana

No exemplo acima, o resultado foi "Moana".

  • Cada elemento do array deve ser separado por vírgula.
  • Os elementos podem ser strings, números, valores booleanos, variáveis, ou qualquer outro tipo de dado. Podem até ser outros arrays ou objetos.

Usando colchetes

A outra forma de declarar um array é semelhante a declarar uma variável, colocando os valores entre colchetes "[]".

Veja:

    let frutas = ['maçã', 'banana', 'laranja'];
    console.log(frutas[1]); // banana

No exemplo acima, o resultado foi "banana".

Trabalhando com os elementos de um Array

Acessando Elementos

Os elementos do array são acessados usando índices, que começam em 0.

    let frutas = ['Maçã', 'Banana', 'Laranja'];
    console.log(frutas[0]); // Maçã
    console.log(frutas[1]); // Banana

Modificando Elementos

Você pode modificar os elementos do array diretamente através de seus índices.

    frutas[1] = 'Manga';
    console.log(frutas[1]); // Manga

Verificar o número de elementos no Array

length: A propriedade length retorna o número de elementos em um array.

    let tamanho = frutas.length;
    console.log(tamanho); // 3

Note que o índice começa do "0", porém o length começa a contagem dos elementos a partir do "1".

Adicionar elementos no final

Para adicionar um ou mais elementos ao final do array e retornar o novo comprimento do array, utilizamos o método push().

    console.log(frutas.push('Morango')); // 4

Note que o comprimento anterior era 3, e agora ele retornou 4 pois adicionou o Morango.

Para adicionar mais de um elemento, basta separá-los com vírgula.

Adicionar elementos no início

Para adicionar um ou mais elementos ao início do array e retornar o novo comprimento do array, basta usar o método unshift().

    console.log(frutas.unshift('Uva')); // 4

Funciona de forma similar ao push(), com a única diferença que o push adiciona no final e o unshift adiciona no início.

Assim como no push(), para adicionar mais de um elemento, basta separá-los com vírgula.

Remover o primeiro elemento

Para remover o primeiro elemento do array e retorná-lo, basta usar o método shift().

    console.log(frutas.shift()); // Maçã

Remover o último elemento do array

Para remover o último elemento do array e retorná-lo, utilizamos o método pop().

    console.log(frutas.pop()); // Morango

Foi testado colocar um número dentro dos parênteses e nada aconteceu, o método excluiu apenas o último elemento.

Descobrindo o índice do elemento

indexOf(): Retorna o índice do elemento especificado, ou -1 se não for encontrado.

    let frutas = ['maçã', 'banana', 'laranja'];
    let indice = frutas.indexOf('banana');
    console.log(indice); // 1

Observação: Caso tenha mais de um elemento igual no array, o indexOf retornará o índice do primeiro elemento encontrado.

Atenção: o "O" de indexOf é maiúsculo.

Copiar array inteiro ou parte

slice(): É usado para copiar uma parte de um array para um novo array sem modificar o array original. Ele aceita dois argumentos opcionais: o índice de início e o índice de fim.

Para isso, esse método usa dois parâmetros:

  1. Número do índice do primeiro elemento que será copiado.
  2. Número do índice do último elemento a ser copiado. Entretanto, esse último não está incluído na cópia. Logo, precisará somar 1 no índice do último elemento que desejar copiar.

O método slice() cria uma cópia superficial do array original. Isso significa que se o array contiver objetos ou outros arrays, somente as referências desses objetos serão copiadas para o novo array. Os objetos em si não são duplicados.

    let frutas = ['uva', 'maçã', 'banana', 'laranja'];
    let novoArray = frutas.slice(1, 3);
    console.log(novoArray); // ['maçã', 'banana']
  • Note que ele copiou somente até o índice 2.
  • Se você fornecer apenas o índice de início, o método slice copiará todos os elementos a partir desse índice, incluindo o próprio, até o final do array.
  • Se usar slice(-2) com valor negativo, ele copiará a quantidade de itens especificados a contar do último item do array.
  • Se usar slice() vazio, ele copiará todo o array.

Forma errada de copiar Arrays:

Se você tentar copiar um array usando array1 = array2, você não estará criando uma cópia do array, mas sim uma referência ao mesmo array. Isso significa que qualquer alteração feita em array2 também afetará array1, e vice-versa.

            let array1 = [1, 2, 3];
            let array2 = array1;
            array2.push(4);
            console.log(array1); // [1, 2, 3, 4]
            console.log(array2); // [1, 2, 3, 4]

No exemplo acima, ao adicionar um elemento em array2, o array1 também foi modificado.

Para criar uma cópia real do array, além do slice(), você também pode usar o operador spread [...array]. Veja o exemplo:

            let array1 = [1, 2, 3];
            let array2 = [...array1];
            array2.push(4);
            console.log(array1); // [1, 2, 3]
            console.log(array2); // [1, 2, 3, 4]

No exemplo acima, ao adicionar um elemento em array2, o array1 permaneceu inalterado.

forEach

Executa uma função em cada elemento do array.

    frutas.forEach(function(item, indice) {
        console.log(item, indice);
    });

No exemplo acima, ele apenas enumerou cada fruta com seu índice.

splice

O método splice() é uma poderosa ferramenta para manipulação de arrays em JavaScript. Ele permite adicionar, remover ou substituir elementos em um array, modificando o array original.

Cuidado para não confundir splice() com slice(). Enquanto slice() cria uma cópia superficial de uma parte do array sem modificar o original, o splice() serve para alterar o array original.

Remover elementos usando splice

Para remover elementos com splice(), utilize a seguinte sintaxe:

  • O primeiro argumento é o índice do elemento a ser removido.
  • O segundo argumento é o número de elementos a serem removidos a partir do índice especificado.

Exemplo:

        let frutas = ['uva', 'maçã', 'banana', 'laranja'];
        frutas.splice(1, 2);
        console.log(frutas); // ['uva', 'laranja']
  • No exemplo acima, 'maçã' e 'banana' foram removidas.
  • Se o segundo argumento for maior que o número de elementos restantes, todos os elementos a partir do índice especificado serão removidos.
  • Se o segundo argumento for 0, nenhum elemento será removido.

Adicionar elementos usando splice

Para adicionar elementos com splice(), utilize a seguinte sintaxe:

  • O primeiro argumento é o índice onde os novos elementos serão adicionados.
  • O segundo argumento é 0 (para não remover nenhum elemento).
  • Os argumentos seguintes são os elementos a serem adicionados.

Exemplo:

        let frutas = ['uva', 'maçã', 'banana', 'laranja'];
        frutas.splice(1, 0, 'kiwi');
        console.log(frutas); // ['uva', 'kiwi', 'maçã', 'banana', 'laranja']
  • No exemplo acima, 'kiwi' foi adicionado na posição 1, deslocando os elementos subsequentes.
  • Podemos adicionar múltiplos elementos separando-os por vírgulas.
  • Se o segundo argumento fosse 1, 'maçã' seria removida e 'kiwi' seria adicionado em seu lugar.
  • Se o segundo argumento fosse maior que 1, os elementos especificados seriam removidos e os novos elementos seriam adicionados na posição indicada.
