Array

Os arrays em JavaScript são estruturas de dados usadas para armazenar múltiplos dados em sequência. Arrays são como variáveis armazenadas em sequência.

Começando sempre do "0"(zero) até o número de elemento que for posto no Array. Essa sequência se chama index do array ou, em português, índice do array.

Declarando Array

Existem duas forma de declarar um array:

Usando o construtor Array 

                            familia = new Array("Allan","Karol","Maya","Moana");
                            console.log(familia[3]);

No caso o resultado foi "Moana"

  • Cada elemento do array deve ser separado por vírgula.
  • Nesse exemplo todos foram strings, mas poderiam ser números, valores booleanos, variáveis, ou qualquer outro tipo que já vimos. Poderia até ser outro array ou um objeto que veremos mais pra frente.

Usando colchetes

A outra forma de declarar um array é semalhante a declarar uma variável. Para isso basta por os valores entre colchetes"[]"

Veja:

    let frutas = ['maçã', 'banana', 'laranja'];
    console.log(frutas[1]); //banana

No caso o resultado foi "banana"

Acessando Elementos

Os elementos do array são acessados usando índices, que começam em 0.

let frutas = ['Maçã', 'Banana', 'Laranja'];
console.log(frutas[0]); // Maçã
console.log(frutas[1]); // Banana

Modificando Elementos

Você pode modificar os elementos do array diretamente através de seus índices.

frutas[1] = 'Manga';
console.log(frutas[1]); // Manga

Verificar quantos elementos tem no Array

length: A propriedade length retorna o número de elementos em um array.

let tamanho = frutas.length;
console.log(tamanho); // 3

Note que o índice começa do ""0"", porém o lentgh começa a contagem dos elementos apartir do "1". Sendo assim 3 elementos.

Métodos Comuns de Arrays

Adicionar elementos no final

Para adicionar um ou mais elementos ao final do array e retorna o novo comprimento do array utilizamos o comando push().

console.log(frutas.push('Morango'));//4

Note que o comprimento anterior era 3, e agora ele retornou 4 pois adicionou o Morango.

Para adicionar mais de um elemento basta separa-los com vírgula.

Adicionar elementos no início

Para adicionar um ou mais elementos ao início do array e retornar o novo comprimento do array basta usar o coamando unshift().

console.log(frutas.unshift('Uva')); //maçã

Funiciona de forma similar ao push(), com a única diferença que o push adiciona no final e esse adiciona no início.

Assim como no push(), para adicionar mais de um elemento basta separa-los com vírgula.

Remover o primeiro elemento

Para remover o primeiro elemento do array e o retornar basta usar o comando shift().

console.log(frutas.shift()); //maçã

Remover o último elemento do array

Para remover o último elemento do array e o retorna utilizamos o comando pop().

console.log(frutas.pop()); //Morango

Foi testado colocar número nesse parenteses e nada aconteceu, excluiu apenas o último mesmo.

Descobrindo o índice do elemento

indexOf(): Retorna o índice do elemento especificado, ou -1 se não for encontrado.

let frutas = ['maçã', 'banana', 'laranja'];
let indice = frutas.indexOf('banana');
console.log(indice);//1

Observação: Caso tenha mais de um elemento igual no array, o indexOf retornará o primeiro. Ou em outras palavras o índice de menor valor.

Atenção o "O" de indexOf é maiúsculo.

Copiar array inteiro ou parte

slice(): É usado para copiar uma parte de um array para um novo array sem modificar o array original. Ele aceita dois argumentos opcionais: o índice de início e o índice de fim.

Para isso esse comando usa duas casas de valores onde a casa:

  1. Número do índice do primeiro elemento que será copiado até o
  2. Número do índice do último elemento a ser copiado.Entretanto esse último não está incluido na cópia. Logo precisará somar 1 no índice do último elemento que desejar copiar .

O método slice() cria uma cópia superficial do array original. Isso significa que se o array contiver objetos ou outros arrays, somente as referências desses objetos serão copiadas para o novo array. Os objetos em si não são duplicados.

let frutas = ['uva', maçã', 'banana', 'laranja'];

let novoArray = frutas.slice(1, 3);
console.log(novoArray)//0:"maçã"
1:"banana"
length:2
  • Note que ele copiou somente até o 2.
  • Se você fornecer apenas o índice de início, o método slice copiará todos os elementos apartir desse índice, incluindo o próprio, até o final do array slice(1);
  • Se usar slice(-2) com valor negativo ele copiará a quantidade de itens especificados a contar do último item do array
  • Se usar slice() vazio ele copiará todo o array;

forEach

Executa uma função em cada elemento do array.

frutas.forEach(function(item, indice) {
console.log(item, indice);
});

No exemplo atual ele apenas enumerou cada fruta com seu índice.

splice

Adiciona/remova elementos de um array.

Remover elemento usando splice

Splice quando usado para remover algum elemento funciona da seguinte forma:

  • O primeiro valor representa o número do índice do elemento do array em questão.
  • O segundo representa quantos elementos você deseja excluir. Lemabrando que ele excluirá a quantidade que escolher apartir do elemento escolhido no primeiro valor.

Exemplo:

    let frutas = ['uva', 'maçã', 'banana', 'laranja'];
    frutas.splice(1, 2);
  • No exemplo acima o programa excluirá maçã e banana
  • Se estivesse colocado um valor maior que a quantidade de item no array, o programa excluiria todos os itens apartir do item selecionado no primeiro valor, inclusive item selecionado.
  • Se o segundo valor for 0 ele excluirá nenhum item. Acontece nada.

Adicionar elemento usando splice

Para adicionar elemento usando splice basta acrescentar um "," e logo em seguida o elemento que seja incluir no array.

let frutas = ['uva', 'maçã', 'banana', 'laranja'];
frutas.splice(1, 0, 'kiwi');
  • No caso acima 'kiwi' ficou no lugar de maçã e maçã passou a ser elemento 2 no index e empurrou todos os posteriores a ele pra uma casa a frente
  • Podemos também adicionar mais elementos basta separar todos por vírgula.
  • Se na segunda casa tivesse o número 1, nesse caso 'maçã' seria excluída e entreria 'kiwi' no lugar dela e nada mais seria modificado.
  • E se fosse algum número superior a 1 ele excluiria assim como visto anteriormente com a única diferença que o programa adicionaria kiwi na posição 1.
