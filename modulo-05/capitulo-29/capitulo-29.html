Java Script Básico

Conhecimentos básicos

Nos capítulos anteriores, vimos que as linguagens HTML e a CSS são fundamentais para a criação de páginas web. O foco do HTML é o conteúdo enquanto o foco do CSS é a formatação dessas páginas.

As linguagens HTML e CSS não são linguagens de programação. Para resolver determinados problemas, é necessário utilizar uma linguagem de programação. Por isso, veremos nesse capítulo, a linguagem de programação JavaScript.

Com a linguagem JavaScript podemos construir páginas extremamente dinâmicas e interativas. O foco do JavaScript é implementar o comportamento ou a inteligência das páginas web

Há duas formas de associar código JavaScript aos documentos HTML.

  • Interno
  • Externo

Para trabalhar com JS é necessário ter intalado o node js no pc para rodar os programas que iremos criar. Para isso basta assistir o vídeo Dando os primeiros passos em JS

Feito isso, existem duas formas de rodar um programa Java script:

  • No VS Code
  • No navegador

Rodando JS no VS Code

Para rodar o Java Script no VS Code basta abrir um novo terminal utlizando a tecla de atalho "ctrl+j" ou abrindo a aba "Terminal" ⇒"Novo terminal"

Após isso abrirá uma janela no VS Code em baixo da janela de código. Nela digita "node" e enter. Pronto agora é so digitar os códigos nela e o resultado será exposto na linha abaixo.

Rodando JS no navegador

Para rodar o JS navegador é necessário associar código JavaScript ao HTML.

Feito isso, abra o navegador ⇒ botão direito na página de navegação⇒ Inpecionar e ⇒ cliquei na aba console. Agora para aparecer os resultados dos código é necessário sempre digitar o coódigo:"console.log(a)"

exemplo:

    var a = 30;
    var b = 5;
    console.log(a+b);

No console irá aparecer apenas o que está entre parenteses. Nesse caso seria 35

JavaScript interno

Assim como o CSS o JS pode ser interno deixando ele amarrado há um único HTML. Nesse caso, diferentemente do CSS, ele fica na última parte do body, para não pesar no carregamento da página, pois diferente do CSS ele não irá gerar efeitos visuais confusos durante o carregamento. O máximo que vai acontecer é não rodar alguns comandos enquanto carrega.

Veja o código abaixo:

Variáveis

Em JavaScript, as variáveis podem ser declaradas usando var, let ou const. Cada uma dessas palavras-chave possui características e comportamentos distintos. Vamos explorar as diferenças entre elas:

Através de qualquer uma dessas duas palavras chaves seguida de seu nome (identificador) criaremos uma variável.

var

Características da variável decalarada com "var":

  • Escopo de Função: A variável declarada com "var" tem escopo de função. Isso significa que se ela for declarada dentro de uma função, não estará acessível fora dessa função.
  • Não tem Escopo de Bloco: "var" não respeita o escopo de bloco. Se você declarar uma variável com "var" dentro de um bloco (como dentro de um "if" ou "for"), ela estará disponível fora desse bloco.
    function testeVar() {
    var x = 1;
    if (true) {
    var x = 2; // Mesmo escopo!
    console.log(x); // 2
    }
    console.log(x); // 2
    }

Note que quando a variável foi chamada pela segunda vez ele estava do escopo da function entretanto ela apresentou o valor do bloco if.

let

Escopo de Bloco: let tem escopo de bloco. Isso significa que se você declarar uma variável com let dentro de um bloco {}, ela só estará disponível dentro desse bloco.

function testeLet() {
let x = 1;
if (true) {
    let x = 2; // Escopo diferente!
    console.log(x); // 2
}
console.log(x); // 1
}

testeLet();

Note que dessa vez, diferentemente, do que houve em "var", agora o resultado do segundo exemplo foi 1, pois a segunda declaração estava no bloco de if. Logo x só é igual a 2 dentro do bloco. if

const

Características da variável decalarada com "const":

  • Escopo de Bloco: Assim como "let", "const" tem escopo de bloco.
  • Imutabilidade: As variáveis declaradas com "const" devem ser inicializadas no momento da declaração e não podem ser reatribuídas. No entanto, se a variável for um objeto ou array, suas propriedades ou elementos podem ser modificados.
const c = 42;
c = 50; // TypeError: Assignment to constant variable.

const obj = { prop: 1 };
obj.prop = 2; // Isto é permitido.
console.log(obj.prop); // 2

Tipos de variáveis

Diferentemente da linguagem C aqui não precisa definir o tipo da variável pois ela se ajustará sozinha de acordo com o valor atribuído a ela. Portanto é importante limitar esse tipo para evitar mudanças desnecessárias e deixar o código pesado em vão.

Existem 3 tipos de varáveis primitivas promordiais em JavaScript que são:

  • Number
  • String
  • Boolean

Identificando o tipo da variável

typeof: Esse comando serve para identificar o tipo que está sendo aplicado àquela variável.

Exemplo:

    'allan'

Note que o tipo de varável se altera de acordo com a necessidade.

Operadores

Para manipular os valores das variáveis de um programa, devemos utilizar os operadores oferecidos pela linguagem de programação adotada. A linguagem JavaScript possui diversos operadores e os principais são categorizados da seguinte forma:

  • Aritmético: + - * / %
  • Atribuição: = += -= *= /= %= ++ --
  • Comparativo: == != < <= > >=
  • Lógico: && ||

Operadores Aritméticos

Os operadores aritméticos funcionam de forma muito semelhante aos operadores da matemática. Os operadores aritméticos são:

  • Adição +
  • Subtração -
  • Multiplicação *
  • Divisão /
  • Módulo % Módulo é o resto de uma divisão. Exemplo: 5 % 4 = 1 pois 5 dividido por 4 sobra 1.

As operações aritméticas em JavaScript obedecem as mesmas regras da matemática com relação à precedência dos operadores e parênteses. Portanto, as operações são resolvidas a partir dos parênteses mais internos até os mais externos, primeiro resolvemos as multiplicações, divisões e os módulos. Em seguida, resolvemos as adições e subtrações.

Concatenação de Strings

String são variávies capazes de suportar conjuntos de caracteres, vulgo frases e assim como no excel é possível concatenar variáveis do tipo string.

Para isso usamos o sinal de "+"Veja um exemplo abaixo: 

    var nome='João'
    var sobrenome='Silva'
    var nomecompleto='nome'+'sobrenome'
    nomecompleto=JoãoSilva

Note que é necessário incluir espaço no nome!

Ao declarar uma string é necessário coloca-la entre aspas duplas ou simples e isso deve ficar padronizado no projeto, pois caso sua string possua aspas no meio do texto, será necessário usar a que não está sendo usada para declarar a string.

Como pudemos notar é o mesmo operador utilidado para adição. Como consequência disso, o JS pode gerar algumas confusões quando precisamos concatenar frases e realizar somas juntas

Exemplo:

    'João'+1 = João1
    'João'+1+1=João11
    1+1+'João'=2João
    1+1+'João'+2+2=2João22
    1+1+'João'+2+2-1=NaN

Como pudemos notar, apartir do momento que usamos um sinal de adição para concatenar, todos os próximos elemento serão reconhecidos como string e todo sinal de adição será reconhecido como concatenação. Caso precise usar algum outro operador aritmético o JS acusará NaN (Not a Number) infomando que não é um número.

Operadores de Atribuição

Como o próprio nome já diz esses sinais servem para atribuir algo do lado direito para algo que está no lado esquerdo do sinal (esse algo é o operando), geralmente o que fica no lado esquerdo são variáveis.O operador de atribuição básico é o igual (=), que atribui o valor do operando à direita ao operando à esquerda. Isto é, x = y atribui o valor de y a x.

Os outros operadores de atribuição são encurtamentos de operadores padrão.

Veja os operadores de distribuição abaixo:

Nome Operador encurtado Significado
Simples = x = y x = y
Atribuição de adição += x += y x = x + y
Atribuição de subtração -= x -= y x = x - y
Multiplicativa * x *= y x = x * y
Divisória /= x /= y x = x / y
Modular %= x %= y x = x % y
Incremento ++ x++ x+1
Decremento -- x-- x-1
OBS.:

Os operadores de Incremento "++" e Decremento "--" podem ser usado antes da variável ficando ++x ou --x. Isso faz diferença. No ++x a variável recebe a soma na hora já no x++ ela recebe a soma somente quando o programa passar para o próximo comando.

Isso acontece porque o navegador carrega na ordem exata que foi digitado então numa suposta ocasião onde o programa encerre em x++ a variável x não irá receber a soma efetivamente.

Que diferença isso faz? Não sei, só sei que é assim e foi citado em aula.

x+5 é diferente de x+=5 pois em x+5 x não altera seu valor apenas apresenta o resultado!

Operadores Comparativos

Esses operadores, como o próprio nome já diz, comparam variaveis ou valores. Eles trabalham com valores booleanos, ou seja, verdadeiro ou falso,true or false.

Operador Descrição Exemplos
Igualdade == Retorna verdadeiro caso os operandos sejam iguais. 3==3 true
Estritamente igual (===) Retorna verdadeiro caso os operandos sejam iguais e do mesmo tipo. Isso é, esse testa o tipo de veriável que está abrigando o valor em questão. 3==="3" false
porém
3=="3" true
Diferença != Retorna verdadeiro caso os operandos não sejam iguais. 3!=3 false 3!=2 true
Estritamente diferente (!==) Retorna verdadeiro caso os operandos sejam diferentes e de tipo diferentes. Isso é, esse testa o tipo de veriável que está abrigando o valor em questão. 3!=="3" true
porém
3!="3" false
Menor que < Retorna verdadeiro caso o operando da esquerda seja menor que o da direita. 3<3 false 3<4 true
Menor ou igual <= Retorna verdadeiro caso o operando da esquerda seja menor ou igual ao da direita. 3<=2 false 3<=3 true
Maior > Retorna verdadeiro caso o operando da esquerda seja maior que o da direita. 3>3 false 3>2 true
Maior ou igual >= Retorna verdadeiro caso o operando da esquerda seja maior ou igual ao da direita. 3>=4 false 3>=3 true

Operadores Lógicos

Esses operadores permitem verificar duas ou mais condições devolvendo valores booleanos.

O JavaScript possui 02 operadores lógicos e eles sempre trabalham verificando o resultado dos operadores comparativos. Os operadores lógico são:

  • && (E): Esse comparador só retornará o valor "true" se todos os operadores comparativos em questão retornarem "true". Caso um retorne "false" esse comparador retronará "false

    • Exemplo:

        10>8 && 10>08 && 10>7 = true
    10>8 && 10>05 && 10>11 = false
  • || (OU): Esse comparador só retornará o valor "False" se todos os operadores comparativos em questão retornarem "false". Caso um retorne "true" esse comparador retronará "true".

    • Exemplo:

        10>8 || 10>15 || 10>7 = true
    10>12 || 10>15 || 10>11 = false

Operador negação "!"

Esse operador é simples e muito utilizado. A função dele é apenas inverter o resultado booleano, se for verdadeiro ele trasforma em falso e se for falso trasforma em verdadeiro.

Para isso basta colocar o sinal de "!" antes de qualquer teste e ele inverterá o resultado daquela teste.

Exemplo:

    !10>5 = false

OBS.: Esse operador pode ser usado apenas no início da linha do teste de resultado booleano, ele não funciona em testes em teste indviduais dentro do teste principal. Exemplo:

10>8 || 10>15 || !10>7 = true

Esse resultado não irá alterar pois e operador de negação não causou efeito

Operador ternário “?:”

O Operador ternário "?" funciona como um if simplificado onde se uma comparação for verdadeira ele irá gerar o primeiro resultado e se for falso irá entregar o segundo resultado.

Exemplo:

    var a = 30;
    var b = 5;
    a==b?"Sim":"Não"

No exemplo acima o resultado será: Não.

Tratamento de dados

Como vimos exitem mais de um tipo de variável em JavaScript, por conta disso, por vezes se faz necessário fazer conversões de tipo, para que o programa consiga tratar os dados da maneira correta.

Transformando string em Number

Existem 3 formas de transformar uma variável string em Number. Mas isso vai dependder se é número inteiro ou fracionado. As duas formas são:

  • Number.parseInt(n): Para número inteiro, exemplo: 1, 2, 3, 4, 5...
  • Number.parseFloat(n): Para número real, exemplo: 1,151; 3,58; 7,5...
  • Number(n): Essa conversão detecta automaticamente o tipo de número que está sendo usado e converte.

Vale ressaltar que obviamente isso só funciona em string do tipo var x="3". Se você escrever "três" o prgrama irá gerar um ERRO.

Number.parseInt(n)

Como já vimos esse comando faz a conversão para número inteiro,

Exemplo 01:

var n1="3";
var n2=4;
var soma=n1+n2;
n1=Number.parseInt(n1);
console.log(soma);//34

Exemplo02:

var n3="3";
var n4=4;
n3=Number.parseInt(n3);
var soma=n3+n4;
console.log(soma);//7
Observações
  • No 1º exemplo o resultado de errado porque a variável soma foi declarada antes da conversão, logo o cálculo foi feito antes da conversão. Mesmo sendo chamada depois, ela apenas exibiu o cálculo ja ja estava feito
  • No segundo exemplo, foi feito o uso correto do comando e a soma foi realizada.
  • Para evitar erros, deve-se converter logo após ter sido criada a variável. Ou fazer a conversão dentro da variável exemplo "var soma=Number.parseInt(n3)+n4;"

Number.parseFloat(n)

O funcionamento desses comandos de conversão são todos iguais. Resta apenas algumas diferenças quanto ao tipo interio ou real.

Exemplo:

{
let n1="7.5";
let n2=9;
let soma=Number.parseFloat(n1)+n2;
console.log(soma);//16.5
}
Observações

A variável está sendo criada em let entre chaves para não precisar ficar trocando o nome da variável em todo exemplo. Como já vimos let não sai do escopo"{}"

Dessa vez a conversão foi feita dentro da variável soma. Assim evita muitos bugs posteriores com as manutenções no código.

Number(n)

Como já foi falado, essa conversão detecta automaticamente o tipo de número que está sendo usado e converte.

Exemplo:

{
    let n1="7.5";
    let n2="10";
    let soma=Number(n1)+Number(n2);
    console.log(soma);//17.5
}

Transformando Number em string

Assim como podemos converter string em number também podemos fazer o contrário. Transformar uma variável tipo number em uma string.

É um pouco menos comum esse uso mas é necessário as vezes. Para isso podemos utilizar 2 comandos diferentes:

  • String(n): Essa é a forma mais simples e funciona de forma similar a number que vimos anteriormente.
  • n.toString(): Tem o mesmo efeito que a anterior, porém a escrita muda.
{
    let n1=7.5;
    console.log(typeof(n1));//number
    n1=String(n1);
    console.log(typeof(n1));//string
}
Atenção para o "S" de "String" que é maiúculo.

Concatenar com crase

Já aprendemos em operadores que para concatenar string utilizamos o sinal de "+". Entertanta existe uma outra forma de cocatenar utilizando crase inves de aspas "".

Essa forma foi lançada na última versão do java script em 2015 e se tornou muito popular por ser menos verbosa.

Para isso usa-se crase no lugar das aspas e sempre que precisar por uma variável no meio da string, basta por a variável entre chaves{} e precedida por um "$". da seguinte forma: ${variável}.

Exemplo comparativo:

{
    let nome="Allan";
    let nota1b=9.5;

    console.log("O aluno "+nome+" ficou com nota "+nota1b+" no 1º bimestre");
    console.log(`O aluno ${nome} ficou com nota ${nota1b} no 1º bimestre`);
}

O resultado foi exatamente o mesmo, mas note que até nesse exemplo com pouca variável no meio do texto. O texto com crase ficou menor.

Formatando Strings

É possível fazer alguma formatações básicas em string utlizando JavaScript e até mesmo tratar os dados de uma string como contar quantos caracteres e string possui.

Conheça agora alguns comandos básicos e suas funcionalidades.

Contar caracteres

.lentgh: lentgh é um atributo ele serve para medir uma string contando quantos caracteres ela possui.

Veja:

{
    let nome="pé sujo";
    console.log(nome.length);//7
}
Note que ele conta os espaços também e não conta os acentos.

Transformando texto em maiúsculo

.toUpperCase(): Esse método transforma todo o texto em maiúsculo.

Veja:

{
    let nome="pé sujo";
    console.log(nome.toUpperCase());//PÉ SUJO
}

Esse diferente é um método note que ele precisa dos parenteses no final diferentemente de length que não precisa. Pois length é um atributo

Transformando texto em minúsculo

.toLowerCase(): Esse método transforma todo o texto em minúsculo.

Veja: 

{
    let nome="PÉ SUJO";
    console.log(nome.toLowerCase());/pé sujo
}

Formatando Números

Casas decimais

.toFixed() Com esse método é possível escolher a quantidade de casas decimais que deseja que apareça depois da vírgula. Seu funcionamento é simples basta por a variável que desja mudar antes do "." e entre os parenteses a quantidade de casas decimais que deseja que apareça.

É interessante usar pelo menos duas casa decimais em unidades de medidas como: metro, kg valores financeiros entre outras.

Exemplo:

{
    let num=2;
    console.log(num.toFixed(2));//2.00
}
