Java Script Básico

Nos capítulos anteriores, vimos que as linguagens HTML e a CSS são fundamentais para a criação de páginas web. O foco do HTML é o conteúdo enquanto o foco do CSS é a formatação dessas páginas.

As linguagens HTML e CSS não são linguagens de programação. Para resolver determinados problemas, é necessário utilizar uma linguagem de programação. Por isso, veremos nesse capítulo, a linguagem de programação JavaScript.

Com a linguagem JavaScript podemos construir páginas extremamente dinâmicas e interativas. O foco do JavaScript é implementar o comportamento ou a inteligência das páginas web

Há duas formas de associar código JavaScript e documentos HTML.

  • Interno
  • Externo

JavaScript interno

Assim como o CSS o JS pode ser interno deixando ele amarrado há um único HTML. Nesse caso, diferentemente do CSS, ele fica na última parte do body, para não pesar no carregamento da página, pois diferente do CSS ele não irá gerar efeitos visuais confusos durante o carregamento. O máximo que vai acontecer é não rodar alguns comandos enquanto carrega.

O código JavaScript pode ser definido dentro de um documento HTML no corpo do elemento script.

Veja o código abaixo:


<body>
.
.
.
    <script>
        alert ("K19");
    </script>
</body>

JavaScript externo

Assim como o CSS o código JavaScript pode ser definido em arquivos separados e depois associados aos documentos HTML. Para isso basta por o atributo src e indicar local do arquivo script.js. Assim como style é recomendo deixar na mesma pasta do index e com o nome script.js

Veja o código abaixo:

<script src =" script .js"> </script>

Variáveis

Assim como qualquer linguagem de programação, JavaScript permite a criação de variáveis. No JS podemos indicar a criação de uma variável de duas formas diferentes:

  • var
  • let

Através de qualquer uma dessas duas palavras chaves seguida de seu nome (identificador) criaremos uma variável. Veja os exemplos abaixo criando uma variável nome e uma variável sobrenome:

                        
var nome;
let sobrenome;

Diferentemente da linguagem C aqui não precisa definir o tipo da variável pois ela se ajustará sozinha de acordo com o valor atribuído a ela. Portanto é importante limitar esse tipo para evitar mudanças desnecessárias e deixar o código pesado em vão.

Existem 3 tipos de varáveis primitivas promordiais em JavaScript que são:

  • Number
  • String
  • Boolean

Typeof

Esse comando serve para identificar que tipo de variável está sendo aplicado àquela variável.

Exemplo:

                        
> let nome="allan"
undefined
> nome
'allan'
> typeof nome
'string'
> nome=1
1
> typeof nome
'number'

Note que o tipo de varável se altera de acordo com a necessidade.

Operadores

Para manipular os valores das variáveis de um programa, devemos utilizar os operadores oferecidos pela linguagem de programação adotada. A linguagem JavaScript possui diversos operadores e os principais são categorizados da seguinte forma:

  • Aritmético: + - * / %
  • Atribuição: = += -= *= /= %= ++ --
  • Relacional: == != < <= > >=
  • Lógico: && ||

Operadores Aritméticos

Os operadores aritméticos funcionam de forma muito semelhante aos operadores da matemática. Os operadores aritméticos são:

  • Adição +
  • Subtração -
  • Multiplicação *
  • Divisão /
  • Módulo % Módulo é o resto de uma divisão. Exemplo: 5 % 4 = 1 pois 5 dividido por 4 sobra 1.

As operações aritméticas em JavaScript obedecem as mesmas regras da matemática com relação à precedência dos operadores e parênteses. Portanto, as operações são resolvidas a partir dos parênteses mais internos até os mais externos, primeiro resolvemos as multiplicações, divisões e os módulos. Em seguida, resolvemos as adições e subtrações.

Operadores de Atribuição

Como o próprio nome já diz esses sinais servem para atribuir algo do lado direito para algo que está no lado esquerdo do sinal (esse algo e o operando), geralmente o que fica no lado esquerdo são variáveis.O operador de atribuição básico é o igual (=), que atribui o valor do operando à direita ao operando à esquerda. Isto é, x = y atribui o valor de y a x.

Os outros operadores de atribuição são encurtamentos de operadores padrão.

Veja os operadores de distribuição abaixo:

Nome Operador encurtado Significado
Simples = x = y x = y
Atribuição de adição += x += y x = x + y
Atribuição de subtração -= x -= y x = x - y
Multiplicativa * x *= y x = x * y
Divisória /= x /= y x = x / y
Modular %= x %= y x = x % y
Incremento ++ x++ x+1
Decremento -- x-- x-1
OBS.:

Os operadores de Incremento "++" e Decremento "--" podem ser usado antes da variável ficando ++x ou --x. Isso faz diferença. No ++x a variável recebe a soma na hora já no x++ ela recebe a soma somente quando o programa passar para o próximo comando.

Isso acontece porque o navegador carrega na ordem exata que foi digitado então numa suposta ocasião onde o programa encerre em x++ a variável x não irá receber a soma efetivamente.

Que diferença isso faz? Não sei, só sei que é assim e foi citado em aula.

x+5 é diferente de x+=5 pois em x+5 x não altera seu valor apenas apresenta o resultado!

Operadores Comparativos

Esses operadores, como o próprio nome já diz, comparam variaveis ou valores. Eles trabalham com valores booleanos, ou seja, verdadeiro ou falso,true or false.

Operador Descrição Exemplos
Igualdade == Retorna verdadeiro caso os operandos sejam iguais. 3==3 true
Estritamente igual (===) Retorna verdadeiro caso os operandos sejam iguais e do mesmo tipo. Isso é, esse testa o tipo de veriável que está abrigando o valor em questão. 3==="3" false
porém
3=="3" true
Diferença != Retorna verdadeiro caso os operandos não sejam iguais. 3!=3 false 3!=2 true
Estritamente diferente (!==) Retorna verdadeiro caso os operandos sejam diferentes e de tipo diferentes. Isso é, esse testa o tipo de veriável que está abrigando o valor em questão. 3!=="3" true
porém
3!="3" false
Menor que < Retorna verdadeiro caso o operando da esquerda seja menor que o da direita. 3<3 false 3<4 true
Menor ou igual <= Retorna verdadeiro caso o operando da esquerda seja menor ou igual ao da direita. 3<=2 false 3<=3 true
Maior > Retorna verdadeiro caso o operando da esquerda seja maior que o da direita. 3>3 false 3>2 true
Maior ou igual >= Retorna verdadeiro caso o operando da esquerda seja maior ou igual ao da direita. 3>=4 false 3>=3 true
