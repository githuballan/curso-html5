Objetos

Objetos em JavaScript são coleções de propriedades, semelhantes aos arrays, mas com algumas diferenças importantes. Enquanto arrays utilizam índices numéricos para acessar seus elementos, objetos utilizam chaves (ou nomes) para acessar suas propriedades.

Por exemplo, em um array, você acessa um elemento pelo seu índice:

    let array = [10, 20, 30];
    console.log(array[0]); // 10

Já em um objeto, você acessa uma propriedade pela sua chave:

    let objeto = { nome: "Allan", idade: 25 };
    console.log(objeto["nome"]); // Allan
    console.log(objeto.idade); // 25

Objetos são mais flexíveis que arrays, pois permitem associar valores a nomes significativos, tornando o código mais legível e fácil de entender. Além disso, objetos podem conter funções, conhecidas como métodos, que permitem definir comportamentos associados ao objeto.

Características de um Objeto

  • Chaves e Valores: Cada propriedade de um objeto tem uma chave (também chamada de nome ou identificador) e um valor. A chave é sempre uma palavra(string), é como se fosse o nome da variável (ou algo que pode ser convertido para string), enquanto o valor pode ser de qualquer tipo, incluindo outros objetos, funções, arrays, números, strings, etc.
  • Sintaxe de Definição: Existem várias maneiras de criar objetos em JavaScript. A forma mais comum é usando a notação literal de objeto, que é representada por chaves {}.
  • Mutabilidade: Objetos em JavaScript são mutáveis, o que significa que você pode alterar suas propriedades após a criação.

Existem algumas formas de criar objetos e uma dela é muito semelhante ao array, entretanto enquanto no array utilizamos colchetes para cria-lo, quando se trata de objetos nós utilizamos chaves

Copiando Objetos com Object.assign

Para copiar um objeto em JavaScript, podemos usar o método Object.assign. Este método copia todas as propriedades enumeráveis de um ou mais objetos de origem para um objeto de destino.

Exemplo:

    let objeto1 = { a: 1, b: 2, c: 3 };
    let objeto2 = Object.assign({}, objeto1);

    console.log(objeto2); // { a: 1, b: 2, c: 3 }

Resultado:

Copiando Objetos com Spread Operator

Outra maneira de copiar um objeto é usando o operador spread (...). Este operador permite criar uma cópia superficial de um objeto.

Exemplo:

    let objeto1 = { a: 1, b: 2, c: 3 };
    let objeto2 = { ...objeto1 };

    console.log(objeto2); // { a: 1, b: 2, c: 3 }

Resultado:

Forma errada de copiar objeto:

Se tentarmos copiar um objeto usando a atribuição direta (objeto1 = objeto2), estaremos apenas criando uma referência ao mesmo objeto, e não uma cópia. Qualquer alteração em um dos objetos afetará o outro.

Exemplo:

    let objeto1 = { a: 1, b: 2, c: 3 };
    let objeto2 = objeto1;
    objeto2.a = 99;

    console.log(objeto1.a); // 99
    console.log(objeto2.a); // 99

Costrutor de objetos trata-se de criar uma função cujo o return seja a criação de um objeto. O objetivo de usar uma função para isso é evitar erros e facilitar a construção de muito objetos com propriedades semelhantes.

Exemplo:

    return{

Desestruturando Objetos

A desestruturação de objetos é uma funcionalidade do ES6 que permite extrair valores de objetos e atribuí-los a variáveis de forma mais concisa e legível. Isso é especialmente útil quando você trabalha com objetos grandes ou complexos.

Exemplo:

            const aluno = { nome: "Allan", idade: 25, curso: "JavaScript" };
            const { nome, idade, curso } = aluno;

            console.log(nome); // Allan
            console.log(idade); // 25
            console.log(curso); // JavaScript

Desestruturando Objetos Aninhados

Você pode desestruturar objetos aninhados diretamente na declaração.

Exemplo:

        const aluno = { nome: "Allan", endereco: { cidade: "São Paulo", estado: "SP" } };
        const { nome, endereco: { cidade, estado } } = aluno;

        console.log(nome); // Allan
        console.log(cidade); // São Paulo
        console.log(estado); // SP

Desestruturando com Alias

Você pode renomear variáveis ao desestruturar objetos.

Exemplo:

        const aluno = { nome: "Allan", idade: 25 };
        const { nome: estudante, idade: anos } = aluno;

        console.log(estudante); // Allan
        console.log(anos); // 25

Desestruturando com Valores Padrão

Você pode definir valores padrão para variáveis ao desestruturar objetos.

Exemplo:

        const aluno = { nome: "Allan" };
        const { nome, idade = 18 } = aluno;

        console.log(nome); // Allan
        console.log(idade); // 18

Principais Utilizações

  • Extração de Dados: Facilita a extração de dados de objetos retornados por APIs ou funções.
  • Parâmetros de Função: Permite desestruturar parâmetros de função para acessar diretamente as propriedades necessárias.
  • Valores Padrão: É possível definir valores padrão para variáveis ao desestruturar objetos.

Objeto Aninhado

Um objeto aninhado é um objeto que contém outro objeto como uma de suas propriedades. Em outras palavras, é um objeto dentro de outro objeto. Isso permite a criação de estruturas de dados mais complexas e hierárquicas, onde cada nível da hierarquia pode conter seus próprios conjuntos de propriedades e métodos.

Exemplo de Objeto Aninhado

Aqui está um exemplo de um objeto aninhado em JavaScript:

        let aluno = {
            nome: "Allan",
            idade: 25,
            endereco: {
                rua: "Rua das Flores",
                numero: 123,
                cidade: "São Paulo",
                estado: "SP"
            },
            notas: {
                matematica: 9,
                portugues: 8,
                ciencias: 7
            }
        };

Neste exemplo, o objeto aluno contém várias propriedades, incluindo nome, idade, endereco e notas. As propriedades endereco e notas são objetos aninhados dentro do objeto aluno.

Acessando Propriedades de Objetos Aninhados

Para acessar as propriedades de um objeto aninhado, você pode usar a notação de ponto (.) ou a notação de colchetes ([]):

        console.log(aluno.nome); // Allan
        console.log(aluno.endereco.cidade); // São Paulo
        console.log(aluno.notas.matematica); // 9

Desestruturando Objetos Aninhados

A desestruturação de objetos aninhados permite extrair valores de objetos aninhados de forma mais concisa e legível:

        const { nome, endereco: { cidade, estado }, notas: { matematica, portugues } } = aluno;
        
        console.log(nome); // Allan
        console.log(cidade); // São Paulo
        console.log(estado); // SP
        console.log(matematica); // 9
        console.log(portugues); // 8

Neste exemplo, as propriedades cidade e estado são extraídas do objeto endereco, e as propriedades matematica e portugues são extraídas do objeto notas, tudo em uma única linha de código.

Utilidade de Objetos Aninhados

Objetos aninhados são úteis para representar dados estruturados e hierárquicos, como informações de um aluno, dados de um endereço, configurações de um aplicativo, entre outros. Eles permitem organizar dados de maneira lógica e acessível, facilitando a manipulação e o acesso às informações necessárias.
