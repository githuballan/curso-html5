Objeto

Introdução

O objeto funciona de forma muito semelhante a um array, pois também é uma colecão de variáveis. Entretanto, quando se trata de objetos chamamos essas variáveis de propriedades do objeto.

Sendo assim, um objeto é uma coleção de propriedades, onde cada propriedade é uma associação entre um nome (chave) e um valor (Assim como as variáveis). Objetos são uma das principais formas de estruturar dados em JavaScript e são extremamente flexíveis.

Características de um Objeto

  • Chaves e Valores: Cada propriedade de um objeto tem uma chave (também chamada de nome ou identificador) e um valor. A chave é sempre uma string (ou algo que pode ser convertido para string), enquanto o valor pode ser de qualquer tipo, incluindo outros objetos, funções, arrays, números, strings, etc.
  • Sintaxe de Definição:Sintaxe de Definição: Existem várias maneiras de criar objetos em JavaScript. A forma mais comum é usando a notação literal de objeto, que é representada por chaves {}.
  • Mutabilidade: Objetos em JavaScript são mutáveis, o que significa que você pode alterar suas propriedades após a criação.
