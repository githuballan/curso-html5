Objeto

O objeto funciona de forma muito semelhante a um array, pois também é uma colecão de variáveis. Entretanto, quando se trata de objetos chamamos essas variáveis de propriedades do objeto.

Sendo assim, um objeto é uma coleção de propriedades, onde cada propriedade é uma associação entre um nome (chave) e um valor (Assim como as variáveis). Objetos são uma das principais formas de estruturar dados em JavaScript e são extremamente flexíveis.

Características de um Objeto

  • Chaves e Valores: Cada propriedade de um objeto tem uma chave (também chamada de nome ou identificador) e um valor. A chave é sempre uma palavra(string), é como se fosse o nome da variável (ou algo que pode ser convertido para string), enquanto o valor pode ser de qualquer tipo, incluindo outros objetos, funções, arrays, números, strings, etc.
  • Sintaxe de Definição:Sintaxe de Definição: Existem várias maneiras de criar objetos em JavaScript. A forma mais comum é usando a notação literal de objeto, que é representada por chaves {}.
  • Mutabilidade: Objetos em JavaScript são mutáveis, o que significa que você pode alterar suas propriedades após a criação.

Criação de Objetos

Existem algumas formas de criar objetos e uma dela é muito semelhante ao array, entretanto enquanto no array utilizamos colchetes para cria-lo, quando se trata de objetos nós utilizamos chaves

Outra diferença é que no array quando precisamos buscar o valor de uma propriedade nós utilizamos o index (que vai do 0 em diante) já no objeto, conseguimos puxar esse valor chamando pela chave da propriedade. Igual fazemos com as variáveis

Exemplo:

                    var aluno={
                        nome:"Allan", 
                        nota1:10, 
                        nota2:9, 
                        nota3:8, 
                        nota4:7
                    };
                    document.write(aluno["nome"]+"<br>");
                    document.write(aluno.nome);

Objeto com array

Também é possível criar Arrays dentro de objetos.

Exemplo:

{
    let aluno={
    nome:"Allan", 
    nota:[10,9,8,7]
    };
document.write(aluno["nome"]+"<br>");
document.write(aluno.nome+"<br>");
document.write(aluno.nota[0]);
}
