Objeto

O objeto funciona de forma muito semelhante a um array, pois também é uma colecão de variáveis. Entretanto, quando se trata de objetos chamamos essas variáveis de propriedades do objeto.

Sendo assim, um objeto é uma coleção de propriedades, onde cada propriedade é uma associação entre um nome (chave) e um valor (Assim como as variáveis). Objetos são uma das principais formas de estruturar dados em JavaScript e são extremamente flexíveis.

Características de um Objeto

  • Chaves e Valores: Cada propriedade de um objeto tem uma chave (também chamada de nome ou identificador) e um valor. A chave é sempre uma palavra(string), é como se fosse o nome da variável (ou algo que pode ser convertido para string), enquanto o valor pode ser de qualquer tipo, incluindo outros objetos, funções, arrays, números, strings, etc.
  • Sintaxe de Definição: Existem várias maneiras de criar objetos em JavaScript. A forma mais comum é usando a notação literal de objeto, que é representada por chaves {}.
  • Mutabilidade: Objetos em JavaScript são mutáveis, o que significa que você pode alterar suas propriedades após a criação.

Criação de Objetos

Existem algumas formas de criar objetos e uma dela é muito semelhante ao array, entretanto enquanto no array utilizamos colchetes para cria-lo, quando se trata de objetos nós utilizamos chaves

Outra diferença é que no array quando precisamos buscar o valor de uma propriedade nós utilizamos o index (que vai do 0 em diante) já no objeto, conseguimos puxar esse valor chamando pela chave da propriedade. Igual fazemos com as variáveis

Exemplo:

                    var aluno={
                        nome:"Allan", 
                        nota1:10, 
                        nota2:9, 
                        nota3:8, 
                        nota4:7
                    };
                    document.write(aluno["nome"]+"<br>");
                    document.write(aluno.nome);

Copiando Objetos com Object.assign

Para copiar um objeto em JavaScript, podemos usar o método Object.assign. Este método copia todas as propriedades enumeráveis de um ou mais objetos de origem para um objeto de destino.

Exemplo:

    let objeto1 = { a: 1, b: 2, c: 3 };
    let objeto2 = Object.assign({}, objeto1);

    console.log(objeto2); // { a: 1, b: 2, c: 3 }

Resultado:

Copiando Objetos com Spread Operator

Outra maneira de copiar um objeto é usando o operador spread (...). Este operador permite criar uma cópia superficial de um objeto.

Exemplo:

    let objeto1 = { a: 1, b: 2, c: 3 };
    let objeto2 = { ...objeto1 };

    console.log(objeto2); // { a: 1, b: 2, c: 3 }

Resultado:

Forma errada de copiar objeto:

Se tentarmos copiar um objeto usando a atribuição direta (objeto1 = objeto2), estaremos apenas criando uma referência ao mesmo objeto, e não uma cópia. Qualquer alteração em um dos objetos afetará o outro.

Exemplo:

    let objeto1 = { a: 1, b: 2, c: 3 };
    let objeto2 = objeto1;
    objeto2.a = 99;

    console.log(objeto1.a); // 99
    console.log(objeto2.a); // 99

Objeto com Métodos (função)

Método nada mais é que função dentro do objeto, assim como as variáveis se chamam propriedades funções se chamam método.

Exemplo:

function calcMedia(...nota)
{
return (this.nota[0]+this.nota[1]+this.nota[2]+this.nota[3])/4;
}

{
var aluno=
{
    nome:"Allan",
    nota:[9,8,9,9],
    media: calcMedia
}
}


document.write(aluno.nome+"<br>");
document.write(aluno.media()+"<br>");

Note o 'this.' utilizado na function, esse this serve para mostrar para a função que o array que ela está buscando está dentro do objeto que a função está sendo aplicada, caso contrário ela poderia buscaria em todo o código e não acharia ou acaharia o errado, pois ela não buscaria dentro do próprio objeto

Vale ressaltar que essa função calcMedia só poderá ser usada como método pois esse this aponta para um array do um objeto se tentar aplica-la fora de um objeto ela dará erro.

Construtor de objeto

Costrutor de objetos trata-se de criar uma função cujo o return seja a criação de um objeto. O objetivo de usar uma função para isso é evitar erros e facilitar a construção de muito objetos com propriedades semelhantes.

Exemplo:

    
function criarAluno(nome,n1,n2,n3,n4){
    return{
        nome:nome,
        n1:n1,
        n2:n2,
        n3:n3,
        n4:n4,
        media:function(){return(this.n1+this.n2+this.n3+this.n4)/4}
    }
}

let turma = [

    criarAluno("Allan",9,9,9,9),
    criarAluno("Joa",3,9,9,9),
]

let aluno=turma[1];

console.log(aluno);
console.log(aluno.media());

document.write("Nome: "+aluno.nome+
    "<br> Nota 1: "+aluno.n1+
    "<br> Nota 2: "+aluno.n2+
    "<br> Nota 3: "+aluno.n3+
    "<br> Nota 4: "+aluno.n4+
    "<br> Média: "+aluno.media()
);
