Uso de varáveis em css

Variáveis é a melhor coisa que inventaram para um programador, e apesar de CSS não ser uma linguagem de programação, ela herdou essa técnica.

Em CSS utlizamos as variáveis para facilitar nossa vida em modificações de projetos. Um da formas mais comuns de uso de variáveis em CSS é com cores e fontes. Pois dessa forma, caso queira trocar a fonte ou a paleta de cores de um site, você não precisará sair trocando todas as cores estilo por estilos. Nessa caso basta atualizar as variáveis definidas em :root.

Veja como fica a criação dela no CSS: 

        :root{
            --cor00: #C5EBD6;
            --cor01: #83E1AD;
            --cor02: #3DDC84;
            --fonte00: Arial, Helvetica, sans-serif;
            --fonte01: 'Times New Roman', Times, serif; 
        }

Veja como fica a utilização dela no CSS: 

        body{
            font-family: var(--font00)
            background-color: var(--cor01)
        }

Atalho:

Se não quiser digitar a variável pode apertar "ctrl+espaço", rolar a barra de rolagem, da janela que abrir, até em baixo e selecionar a variável que deseja aplicar.

As variáveis personalizadas em CSS devem ter seus nomes iniciando com dois traços obrigatoriamente.

Note que as varáveis estão com nome de cor01, cor00 e nem o nome da fonte. Isso é porque se colocarmos os nomes e percisar mudar futuramente teriamos algo do tipo:

--corverde: #ffffff; (branco) --fontearial: Times New Roman';

Tampouco devemos usar o nome de onde usaremos, pois se usarmos em mais de um elemento ficará uma bagunça.

Responsividade

Responsividade é a capacidade que um site tem de se adaptar a diferentes tamanho de tela como: desde a tela de uma televisão até a tela de um celular.

Uma das técnicas básicas que podem ser utilizadas é deixar algumas caixas estrategicamente um pouco mais flexíveis indicando valores mínimos (min-width / min-height) e máximos (max-width / max-height).

Esses valore impedem que a box cresça demais e fique ruim de ler em uma tela grande como texto muito espalhado.E impede também que reduza muito a ponto não conseguirmos ler textos de tão curta que é a tela.

Responsividade de Imagem

Já vimos anteriormente o uso da tag picture com a tag source:media para carregar imagens de acordo com o tamanho da tela.

Porém, a imagem não ficava perfeita, ela trocava a imagem mas ficava um intervalor entre uma e outra gerando a barra lateral. para evitar esse tipo de problema podemos usar o estilo: width: 100%;

Exemplo em head: 

        img{
            width: 100%;
        }
Assim todas as imagens ocuparão 100% da largura da tela e ela irão se adaptando.

OBS.: Box de textos como header, main, p e etc, já fazem isso normalmente. Nesses boxes de textos a única inportância é definirmos as larguras máximas e mínimas.

Entretanto, ainda assim é necessário usar a tag picture com a tag source:media para que o site não fique com carregamento lento em aparelhos menores.

Atenção

É importante sempre definir a larguara máxima e das imagens usando o style inline.
Pois imagine uma imagem com resolução de 600px de largura, abrindo numa tela de 1080, a imagem ficaria com uma péssima qualidade.

É possível aplicar outras técnicas adicionais para dar ainda mais capacidade de adaptação do conteúdo, como as media queries com a regra @media e as caixas flexíveis com as flex boxes. Mais pra frente, abordaremos esses conteúdos no curso.

Uso do seletor "*"

Basicamente ele aplica uma configuração padrão para TODOS os elementos do código HTML ao qual o estilo está sendo aplicado.

Por Exemplo pode usar esse seletor para remover todas as margens de todas as boxes. Isso é util para evitar erros em projetos pois dessa forma o site fica inteiro nas nossas mãos. Sem pré configurações nos atrapalhando.

Espaçamento entrelinhas em Textos

Outra configuração muito importante que podemos fazer para textos muito longos é o de espaçamento entre as linhas do nosso texto.
Usando a propriedade line-height podemos dizer qual é o tamanho do espaço entre uma linha e a outra do texto. O valor padrão na maioria dos navegadores é algo próximo ao 1.2em

Exemplo comum:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Mollitia culpa recusandae eveniet ad nobis unde molestiae, ut aperiam quisquam beatae in, excepturi nulla! Ut perspiciatis alias id quod? Culpa, qui?

Sombras em Textos

No capítulo anterior, falamos sobre as sombras em caixas, utilizando a propriedade box-shadow. Pois saiba que também existe uma propriedade específica para criar sombras em textos: o text-shadow. A propriedade text-shadow também pode ter quatro parâmetros principais:

1. Deslocamento horizontal (h-offset): quanto a sombra vai andar para o lado direito (valores negativos causam deslocamento para a esquerda)

2. Deslocamento vertical (v-offset): quanto a sombra vai andar para baixo (valores negativos causam deslocamento para cima)

3. Embaçamento (blur): quanto a sombra vai se espalhar pelo fundo

4. Cor (color): cor da sombra. É possível usar transparência

Margem Negativa

A margem negativa é usada para puxar os elementos em volta pra dentro dela, fica como uma espécie de buraco negro e pode ser usada para criar encaixes de box. Porém se for apenas estético também pode mos usar o outiline, tudo vai depender do contexto.

Veja o exemplo abaixo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Ducimus laudantium est quam maiores. Atque accusamus molestias repellat deserunt culpa placeat ducimus alias, suscipit autem corporis aut eius quasi sapiente hic?

Veja o código desse exmplo: 

<div style="background-color: aquamarine; padding: 10px;">
        <p style="background-color: antiquewhite; margin: -10px;">
              Lorem ipsum dolor sit amet consectetur adipisicing
               elit. Ducimus laudantium est quam maiores. Atque 
               accusamus molestias repellat deserunt culpa 
               placeat ducimus alias, suscipit autem corporis aut
                eius quasi sapiente hic?
        </p>
    </div>
