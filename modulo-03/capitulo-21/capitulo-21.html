Tabelas

Tabelas em html são 100% desenhada por comandos, porém de forma muito parecida com as listas.

O comando usado para fazer tabela é o table assim como o ul de lista. Porém temos ainda o tr e o td.

  • tr: o tr representa a linha da tabela cada tr equivale a uma linha e dentro do tr vai o td (fica igual a lista)
  • td: Cada td representa uma coluna da linha que ele está incluso e esse td é como o li da lista. Nele vai conter os dados da célula

O que define a quantidade de linha é a quantidade de tr e o que define a quantidade de coluna é o tr que possuir mais td.

Exemplo:

A1 B1 C1 C1
A2 B2 C2 C2
A3 B3 C3 C3
A4 B4 C4 C4

O comando fica assim: 

<table>    
    <tr>
        <td>A1</td>
        <td>B1</td>
        <td>C1</td>
        <td>C1</td>
    </tr>
    <tr>
        <td>A2</td>
        <td>B2</td>
        <td>C2</td>
        <td>C2</td>
    </tr>
    <tr>
        <td>A3</td>
        <td>B3</td>
        <td>C3</td>
        <td>C3</td>
    </tr>
    <tr>
        <td>A4</td>
        <td>B4</td>
        <td>C4</td>
        <td>C4</td>
    </tr>
</table>

A w3c permite que não encerre a tag tr com </tr> nem e td com </td>. Porém é melhor encerrar para evitar erros.

Tabela Semântica

O objetivo de trabalhar com trabalhar com tabelas semânticas é organizar melhor sua tabela. A w3c recomenda o uso desse tipo sempre que for usar tabelas grandes, com muitos dados. No entanto é bom usar essa tipo de tabela sempre que possível, pois o google valoriza essa Semântica.

São poucas diferenças entre esta e a anterior. Esta possui thead(cabeçalho), tbody(corpo) tfoot(Rodapé). Essa diferença serve apenas para que o programa não misture informação de dados com tiítulos e compilados.

Outro elemento que devems mencionar é o capition. Esse funciona como o título da tabela. (OBS.: Ele fica acima do thead pois no thead fica o título da coluna, já o caption é o título da tabela)

Temos ainda o th, esse vai no lugar de td em locais de Títulos:

  • th: table head(Títulos da tabela)
    • O th não precisa estar somente em thead, avezes ele poderá estar também em tbody ocupando a primeira coluna.
  • td: table data(Dados da tabela)

Scope

Scope é um atributo semântico, ele serve apenas para informar se o th (título) é de uma informação da linha ou da coluna. É muito importante esse tipo de informação para um SEO melhor.

  • scope="col" usado para th de coluna
  • scope="colgroup" usado quando o th está mesclado e abrange mais de uma coluna
  • scope="row" usado para th de linha
  • scope="rowgroup" usado quando o th está mesclado e abrange mais de uma linha

Exemplo: de tabela semântica:

Caption
Peixe PH min PH max Valor
Neon 4,0 6,5 2
Acará Bandeira 4,0 7,2 16
Rodostumus 4,0 6,8 3
Ramirezi 4,0 6,8 20
PH Limite 4,0 6,5 41

<table>
        <caption>Caption</caption>
        <thead>
            <tr>
                <th scope="col">Peixe</th>
                <th scope="col">PH min</th>
                <th scope="col">PH max</th>
                <th scope="col">Valor</th>
            </tr>

        </thead>

        <tbody>
            <tr>
                <td>Neon</td>
                <<td>4,0</td>
                <td>6,5</td>
                <td>2</td>
            </tr>
            <tr>
                <td>Acará Bandeira</td>

Efeito zebrado

O Efeito zebrado é feito pela pseudo-classe ":nth-child". Esse pseudo classe tem muitas opções de configurações veja abaixo:

  • :nth-child(2n) Assim ela pinta somente a 2ª linha de cada 2, em outras palavras, linha sim linha não.
  • :nth-child(3n) Assim ela pinta somente a 3ª linha de cada 3 ( Esse n funciona da mesma forma para qualquer valor de n)
  • :nth-child(3n-1) Assim ela pinta somente a 2ª linha de cada 3 ( Ele separa 3 linha retorna uma e pinta Mas não fica igual o 2n porque ele separa de 3 em 3 logo ficará duas juntas sem pintar e uma pintada). Esse recurso é pouco usado e pode ser aplica com outros valores.
  • :nth-child(ods) (ods = Impar em inglês) Assim ela pinta somente linhas impares.
  • :nth-child(even) (even = Par em inglês) Assim ela pinta somente linhas pares.

Mesclagem de células

Mesclagem de célula é bem simples, para isso usaremos dois atributos que serão adicionados ao td ou th. Veja os atributos abaixo:

  • colspan="2" Nesse atributo ele mesclaria duas colunas , a atual e a próxima a direita. SE quisesse poderia mesclar mais também só aumentar o número
  • rowspan="3" Nesse atributo ele mesclaria 3 linhas , a atual e as próximas duas abaixo. SE quisesse poderia mesclar mais também só aumentar o número

Exemplo:

A1/B1/C1 D1
A2 B2 C2/C3 D2
A3 B3 D3
A4 B4 C4 D4

Comando: 

    <table>    
        <tr>
            <td colspan="3">A1/B1/C1</td>
           

            <td>D1</td>
        </tr>
        <tr>
            <td>A2</td>
            <td>B2</td>
            <td rowspan="2">C2/C3</td>
            <td>D2</td>
        </tr>
        <tr>
            <td>A3</td>
            <td>B3</td>
            
            <td>D3</td>
        </tr>
        <tr>
            <td>A4</td>
            <td>B4</td>
            <td>C4</td>
            <td>D4</td>
        </tr>
    </table>

Tag para Colunas

Como já vimos anterior é possível criar uma tabela zebrada usando a pseudo-classe :nth-child(even). Ou podemos ainda fazer diretamente na linha usando style na tag tr. Bem simples

Mas quando o assunto é coluna precisamos mudar td por td o que é inviável. Por isso existe a tag <colgroup>.

Essa tag funciona da seguinte forma:

Comando no body: 

        <table>  
        
            <colgroup>
                    <col class="A">
                    <col class="B">
                    <col class="C">
                    <col class="D">
            </colgroup>
        

            <tr>
                <td>A1</td>
                <td>B1</td>
                <td>C1</td>
                <td>C1</td>
            </tr>
            <tr>
                <td>A2</td>
                <td>B2</td>
                <td>C2</td>
                <td>C2</td>
            </tr>
            <tr>
                <td>A3</td>
                <td>B3</td>
                <td>C3</td>
                <td>C3</td>
            </tr>
            <tr>
                <td>A4</td>
                <td>B4</td>
                <td>C4</td>
                <td>C4</td>
            </tr>
        </table>

Comandono head: 

        
            col.A{
                background-color: aquamarine;
            }
            col.B{
                background-color: blueviolet;
            }
            col.C{
                background-color: rgb(103, 134, 201);
            }
            col.D{
                background-color: rgb(206, 214, 157);
            }

A4 B4 C4 C4

Atributo para col

Assim com td e th possuem colspan ou rowspan para mesclar células. Em col temos o atributo span que serve para usar o mesmo efeito em uma ou mais colunas ao lado direito. o comando é simples: 

    <table>
            
                <colgroup>
                        <col class="A">
                        <col class="B" span="2">
                        
                        <col class="D">
                </colgroup>

Nesse caso acima o efeito da coluna "B" expandiu para a "C" com isso foi necessário excluir ela.

OBS.: Caso esqueça de excluir um ao usar span o programa empurraria todos os estilos e o seu último estilo configurado ficaria sem efeito.

Tabela responsiva

Criar tabelas responsivas é bem fácil de fazer. Basicamente precisa por a tabela em uma div e uma class para essa div(normal) e em style no head colocar overflow-x: auto para essa div. Veja um exemplo abaixo:

A1 B1 C1 D1 E1 F1 G1 I1 J1 K1 L1
A2 B2 C2 D2 E2 F2 G2 I2 J2 K2 L2
A3 B3 C3 D3 E3 F3 G3 I3 J3 K3 L3
A4 B4 C4 D4 E4 F4 G4 I4 J4 K4 L4

Comando body: 

<div class="responsiva">
        <table>
        .
        .
        .
        </table>
    </div>

Comando Head: 

<style>
            div.responsiva{
                width: 500px;
                overflow-x: auto;
            }
    </style>
OBS.: Essa width foi só para exemplificar como fica sem precisar reduzir a tela, mas se reduzir ela se adapta.
