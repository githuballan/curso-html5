Capitulo 19 - Imagens de Fundos

Podemos usar imagens como Plano de Fundo em qualquer elemento caixa desde "body" até uma simples "div". Para isso, usamos a Propriedade de Estilo " background-image " (a mesma usada para degradê) com a função " url "... Após isso é só inserir o endereço da imagem dentro dos (parênteses). Exemplo: 

    
    div {
            background-image: url(imagens/pattern003.png);
        }

Podemos ainda adicionar uma imagem de outro site como plano de fundo do nosso site. Para isso basta adicionar a URL da imagem no local da URL da imagem. Mas cuidado, caso a imagem do link saia do ar e deixe de existir, isso irá prejudicar o visual do seu site, visto que não teremos mais acesso à essa imagem.

Lembre-se, muito cuidado com o tamanho das imagens de fundo que você inserir.

Funções background-repeat

De forma PADRÃO, imagens de fundo sempre serão carregadas no seu tamanho/dimensão total. Se a imagem de fundo que inserirmos tiver uma dimensão menor do que o respectivo elemento caixa, ela simplesmente será introduzida de forma "repetida", em blocos, para conseguir cobrir toda a área. Já se essa imagem tiver uma dimensão muito grande para o elemento caixa, veremos apenas um pedaço da mesma como background.

Entretanto, podemos usar a função no-repeat para que ela não se repita até cobrir toda a página porém ela ficará do tamnho máximo dela canto superior esquerdo da box. por isso é importante sempre deixar a box no tamanho máximo da imagem.

Exemplo no head:

    <style>    
    div {
            background-image: url(imagens/pattern003.png);
            background-repeat: no-repeat ;
        }
        </style>

Temos ainda alguns efeitos menos usados como o repeat-x, que repete as imagens somente na horizontal(eixo x) ou repeat-y que repete somente no sentido vertical (eixo y).

Exemplo no head:

    <style>    
    div {
            background-image: url(imagens/pattern003.png);
            background-repeat: repeat-x;
        }
        </style>

Background-position

Também podemos mudar a posição de referência de início das repetições. Por padrão, é considerado o canto esquerdo superior (left top), mas podemos ter várias opções. Use a imagem abaixo como referência sempre que precisar definir a posição do fundo com a propriedade background-position no seu código.

Exemplo:

Exemplo no head:

        <style>    
        div {
                background-image: 

                url(imagens/pattern003.png);
                background-repeat: repeat-x;
                background-position: center center;
            }
            </style>

Background-size

Outra coisa que podemos fazer é redimensionar a imagem para forçá-la a caber na caixa. Por padrão, nenhum redimensionamento será aplicado, e a imagem será exibida do seu tamanho natural. Porém, podemos usar a propriedade background-size para alterar esse comportamento.

Os valores aceitos por essa propriedade são:

Exemplo no head:

    <style>    
    div {
            background-image: url(imagens/pattern003.png);
            background-size: cover;
        }
        </style>

Background-attachment

A última propriedade que podemos configurar é o vínculo (attachment) da imagem de fundo com o resto do documento, principalmente se o conteúdo for maior do que a altura da página e seja necessário vazar uma rolagem vertical.

A propriedade background-attachment aceita os valores:

Exemplo no head:

        <style>    
        div {
                background-image:
                url(imagens/pattern003.png);
                background-size: cover;
                background-attachment: fixed;
            }
            </style>

Background Shothands

Como percebemos Background tem muita configuração. Sendo assim, existe uma shothand para configuração completa do Background em apenas uma linha. A propriedade abreviada background pode ser declarada agrupando as seguintes configurações e nessa ordem:

  1. background-color
  2. background-color
  3. background-position
  4. background-repeat
  5. background-size
  6. background-attachment

Use o background-size separado pois não está funcionando no Shothands.ALERTADO PELO PRÓPRIO GUANBARA EM VÍDEO

Exemplos no head:

    <style>    
    div {
            background: 
            black url(imagens/pattern003.png)
             center center no-repeat fixed; 
            background-size: cover;
        }
        </style>

Centralizando Box vertical e horizontal

Como já tentamos o posicionamento de caixas é bem difícil. Poré. existem 2 comandos chaves para isso:

  • position: relative; a box externa deve sempre ser mantida com a position: relative; pois ela está avisando para as box interna que o posicionamento delas agora será em relação a ela e não ao body
  • position: absolute; A box interna deve assumir essa position, pois ess position permite se posicionar de foma absoluta, como os comandos left e top em relação a box relative onde ela se encontra.
  • transform: translate: Com esse comando transform é possivel faz mil modifiações na box. porém agora usaremos apenas o translate, pois esse permite modificar o a base da box que naturalmente vem como superior esquerda

O translate modificar a base da box tanto para fora dela como para dentro. Como sempre para dentro os valores são negativos.

Logo para puxar a base da box da superior esquerda para o centro basta reduzir 50% da esquerda e 50% de cima ficando assim:

Exemplo:

transform: translate(-50%, -50%);

Veja abaixo todos os comnados usado no head para fazer as box azul centralizar com o centro da amarela: 

    div#fora{
        text-indent: 0px;
        height: 400px;
        width: 400px;
        background-color: yellow;
        padding: 0px;
        position: relative;

    }
    div#interna{
       
        height: 150px;
        width: 150px;
        background-color: blue;
        margin: auto;

        position: absolute;
        left: 50%;
        top: 50%;

        transform: translate(-50%, -50%);

    }
