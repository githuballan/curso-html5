Efeito Parallax

Efeito Paralax simula uma sensação de você está passando por uma janela, e conforme vai mudando o ângulo de visão você vai conseguindo enxergar mais da paisagem.

Veja o efeito Parallax abaixo:

Não funciona se usar o transform: translate pois perde o efeito Paralax

Veja o código usado:

                <div style="
            display: inline-block;
            width: 400px;
            height: 400px;
            margin-left: 25vw;
            background-image: url(imagens/wallpaper002.jpg);
            background-attachment: fixed;
            background-size: cover;
            box-shadow: inset 3px 3px 3px 5px rgba(0, 0, 0, 0.295);
                    ">
                    </div>

O segredo para esse efeito está em inserir uma background-image com o background-attachment: fixed em uma box que está dentro de outra box maior, que nesse caso foi a box main.

Assim a imagem fica fixa porém o box fica correndo com a tela
